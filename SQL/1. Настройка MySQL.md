Запускаем MySQL command line и вводим создаем новую базу данных

```SQL
create database test;
use test;
create table users (id integer auto_increment primary key, name varchar(30), age integer)
```

Добавим первую запись в нашу таблицу:

```SQL
insert into users (name, age) values ('Tom', 34);
```

А теперь получим данные из этой таблицы:

```SQL
select * from users;
```

Теперь в MySQL Workbench создаем нвашу базу данных:

```SQL
CREATE DATABASE userdb;
```

Теперь напишем скрипт, который заполнить нашу бд:

```SQL
USE userdb;
CREATE TABLE users (
	id INTEGER AUTO_INCREMENT PRIMARY KEY,
	firstname VARCHAR(30),
	age INTEGER
);
INSERT INTO users (firstname, age) VALUES ('Tom', 34)
```
