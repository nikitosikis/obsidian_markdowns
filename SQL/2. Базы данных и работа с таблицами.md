		# Создание и удаление баз данных

Для того, чтобы создать базу данных используется команда `СREATE DATABASE`
Со следующим синтаксисом:

```SQL
CREATE DATABASE [IF NOT EXIST] database_name;
```
 
 `[IF NOT EXIST]`  - опция, которая исполняет команду, если базы данных с указанным именем еще не существует.

Для того, чтобы установить базу данных в качестве используемой в данный момент применяется команда:

```SQL
USE database_name;
```

Для удаления базы данных используется команда:

```SQL
DROP DATABASE [IF EXIST] database_name;
```

# Создание и удаление таблиц

Для создания таблиц используется команда `CREATE TABLE`, она имеет следующий формальный синтаксис:

```SQL
CREATE TABLE database_name
(
first_column data_type column_atributes,
second_column data_type column_atributes,
...
table_level_atributes
)
```

Создадим пример таблицы:

```SQL

CREATE DATABASE productsdb;

USE productsdb;

CREATE TABLE Customers;
(
	Id INT,
	Age INT,
	FirstName varchar(20),
	LastName varchar(20)
);
```

Для переименования таблицы используется команда:

```SQL
RENAME TABLE old_name TO new_name;
```

Для очистки таблицы используется команда:

```SQL
TRUNCATE TABLE database_name;
```

Для полного удаления таблицы используется команда:

```SQL
DROP TABLE database_name;
```

# Изменение таблиц и столбцов

Если таблица была уже ранее создана, и ее необходимо изменить, то для этого применяется команда ALTER TABLE

Для добавления нового столбца используется команда ADD:

```SQL
ALTER TABLE Customers
ADD Adress VARCHAR(50) NULL;
```

Для удаления столбца:

```SQL
ALTER TABLE Customers
DROP COLUMN Address;
```

Изменение значения по умолчанию:

```SQL
ALTER TABLE Customers
ALTER COLUMN Age SET DEFAULT 22;
```

Изменение типа столбца:

```SQL
ALTER TABLE Customers
MODIFY COLUMN FirstName CHAR(100) NULL;
```

Добавление и удаление внешнего ключа:

```SQL
CREATE TABLE Customers(
	Id INT PRIMARY KEY AUTO_INCREMENT,
	Age INT,
	FirstName VARCHAR(20) NOT NULL,
	LastName VARCHAR(20) NOT NULL,
	Phone VARCHAR(20) NOT NULL UNIQUE
);

CREATE TABLE Orders(
	Id INT PRIMARY KEY AUTO_INCREMENT,
	CustomerId INT,
	CreatedAt DATE
);

ALTER TABLE Orders
ADD CONSTRAINT Orders_customers_fk
ADD FOREIGN KEY(CustomerId) REFERENCES Customers(Id);

ALTER TABLE Orders
DROP FOREIGH KEY Orders_customers_fk;
```

Добавление и удаление первичного ключа:

```SQL
CREATE TABLE Products(
	Id INT,
	Model VARCHAR(20)
);

ALTER TABLE Products
ADD PRIMARY KEY ID;

ALTER TABLE Products
DROP PRIMARY KEY;
```
