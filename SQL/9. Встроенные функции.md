## 6.1. Функции для работы со строками

Строковые функции нужны, чтобы объединять строки, менять регистр, обрезать пробелы, искать подстроки, заменять части текста и получать фрагменты строки.

### Объединение строк

```sql
SELECT CONCAT('Tom', ' ', 'Smith');
-- Tom Smith
```

`CONCAT()` объединяет несколько значений в одну строку. В неё можно передавать строки, числа и даты.

```sql
SELECT CONCAT_WS(' ', 'Tom', 'Smith', 'Age:', 34);
-- Tom Smith Age: 34
```

`CONCAT_WS()` тоже объединяет строки, но первым аргументом принимает разделитель.

---

### Длина строки

```sql
SELECT LENGTH('Tom Smith');
-- 9
```

`LENGTH()` возвращает длину строки.

---

### Удаление пробелов

```sql
SELECT LTRIM('  Apple');
SELECT RTRIM('Apple   ');
SELECT TRIM('  Tom Smith   ');
```

- `LTRIM()` — удаляет пробелы слева;
- `RTRIM()` — удаляет пробелы справа;
- `TRIM()` — удаляет пробелы с двух сторон.

Можно уточнить направление удаления:

```sql
SELECT TRIM(BOTH FROM '  Tom Smith   ');
SELECT TRIM(LEADING FROM '  Tom Smith   ');
SELECT TRIM(TRAILING FROM '  Tom Smith   ');
```

---

### Поиск подстроки

```sql
SELECT LOCATE('om', 'Tom Smith');
-- 2
```

`LOCATE(find, search)` ищет позицию подстроки `find` в строке `search`.

Если подстрока не найдена, возвращается `0`.

```sql
SELECT LOCATE('mig', 'Tom Smith');
-- 0
```

Также можно задать позицию, с которой начинать поиск:

```sql
SELECT LOCATE('m', 'Tom Smith', 4);
-- 6
```

---

### Получение части строки

```sql
SELECT LEFT('Apple', 3);
-- App

SELECT RIGHT('Apple', 3);
-- ple
```

- `LEFT(str, count)` берёт указанное количество символов слева;
- `RIGHT(str, count)` берёт указанное количество символов справа.

```sql
SELECT SUBSTRING('Galaxy S8 Plus', 8);
-- S8 Plus

SELECT SUBSTRING('Galaxy S8 Plus', 8, 2);
-- S8
```

`SUBSTRING(str, start, length)` возвращает часть строки, начиная с позиции `start`. Параметр `length` необязательный.

```sql
SELECT SUBSTRING_INDEX('Galaxy S8 Plus', ' ', 1);
-- Galaxy

SELECT SUBSTRING_INDEX('Galaxy S8 Plus', ' ', 2);
-- Galaxy S8

SELECT SUBSTRING_INDEX('Galaxy S8 Plus', ' ', -2);
-- S8 Plus
```

`SUBSTRING_INDEX(str, delimiter, count)` возвращает часть строки до указанного количества разделителей. Если `count` положительный — берёт с начала, если отрицательный — с конца.

---

### Замена и вставка текста

```sql
SELECT REPLACE('Galaxy S8 Plus', 'S8 Plus', 'Note 8');
-- Galaxy Note 8
```

`REPLACE(search, find, replace)` заменяет одну подстроку на другую.

```sql
SELECT INSERT('Galaxy S9', 8, 3, 'Note 9');
-- Galaxy Note 9
```

`INSERT(str, start, length, insert)` вставляет строку `insert` в строку `str`, заменяя `length` символов с позиции `start`.

---

### Регистр и переворот строки

```sql
SELECT LOWER('Apple');
-- apple

SELECT UPPER('Apple');
-- APPLE

SELECT REVERSE('123456789');
-- 987654321
```

- `LOWER()` переводит строку в нижний регистр;
- `UPPER()` переводит строку в верхний регистр;
- `REVERSE()` разворачивает строку наоборот.

---

### Повторение и дополнение строки

```sql
SELECT REPEAT('ab', 5);
-- ababababab
```

`REPEAT(str, count)` повторяет строку заданное количество раз.

```sql
SELECT LPAD('Tom Smith', 13, '*');
-- ****Tom Smith

SELECT RPAD('Tom Smith', 13, '*');
-- Tom Smith****
```

- `LPAD(str, length, pad)` дополняет строку слева;
- `RPAD(str, length, pad)` дополняет строку справа.

---

### Пример с таблицей

```sql
SELECT 
    UPPER(LEFT(Manufacturer, 2)) AS Abbreviation,
    CONCAT(ProductName, ' - ', Manufacturer) AS FullProdName
FROM Products
ORDER BY Abbreviation;
```

Здесь:

- из названия производителя берутся первые 2 символа;
- они переводятся в верхний регистр;
- название товара и производителя объединяются в одно поле.

---

## 6.2. Функции для работы с числами

Числовые функции нужны для округления, получения модуля, степени, корня, знака числа и случайных значений.

### Округление

```sql
SELECT ROUND(1342.345, 2);
-- 1342.35
```

`ROUND(number, length)` округляет число. Если `length` положительное, округление идёт до указанного количества знаков после запятой.

```sql
SELECT ROUND(1342.345, -2);
-- 1300
```

Если `length` отрицательное, округление идёт до разряда слева от запятой.

---

### Отбрасывание дробной части

```sql
SELECT TRUNCATE(1342.345, 2);
-- 1342.34
```

`TRUNCATE(number, length)` не округляет, а просто оставляет нужное количество знаков после запятой.

---

### Модуль числа

```sql
SELECT ABS(-123);
-- 123
```

`ABS()` возвращает абсолютное значение числа.

---

### Округление вверх и вниз

```sql
SELECT CEILING(123.45);
-- 124

SELECT FLOOR(123.45);
-- 123
```

- `CEILING()` возвращает ближайшее целое число, которое больше или равно исходному;
- `FLOOR()` возвращает ближайшее целое число, которое меньше или равно исходному.

Для отрицательных чисел важно помнить направление:

```sql
SELECT CEILING(-123.45);
-- -123

SELECT FLOOR(-123.45);
-- -124
```

---

### Степень и квадратный корень

```sql
SELECT POWER(5, 2);
-- 25

SELECT POWER(5, 3);
-- 125
```

`POWER(number, degree)` возводит число в степень.

```sql
SELECT SQRT(225);
-- 15
```

`SQRT()` возвращает квадратный корень.

---

### Знак числа

```sql
SELECT SIGN(-5);
-- -1

SELECT SIGN(7);
-- 1

SELECT SIGN(0);
-- 0
```

`SIGN()` показывает знак числа:

- `-1`, если число меньше нуля;
- `1`, если число больше нуля;
- `0`, если число равно нулю.

---

### Случайное число

```sql
SELECT RAND();
```

`RAND()` возвращает случайное число от `0` до `1`.

---

### Пример с таблицей

```sql
SELECT ProductName, ROUND(Price * ProductCount, 2) AS TotalPrice
FROM Products;
```

Запрос считает общую стоимость товара и округляет результат до двух знаков после запятой.

## 6.3. Функции для работы с датами и временем

Эти функции позволяют получать текущую дату и время, извлекать части даты, форматировать дату и прибавлять/вычитать интервалы.

### Текущая дата и время

```sql
SELECT NOW();
SELECT SYSDATE();
SELECT CURRENT_TIMESTAMP();
```

Эти функции возвращают текущую дату и время.

```sql
SELECT CURDATE();
SELECT CURRENT_DATE();
```

Возвращают текущую дату.

```sql
SELECT CURTIME();
SELECT CURRENT_TIME();
```

Возвращают текущее время.

```sql
SELECT UTC_DATE();
SELECT UTC_TIME();
```

Возвращают дату и время относительно UTC/GMT.

---

### Извлечение частей даты

```sql
SELECT DAYOFMONTH('2018-05-25');
-- 25

SELECT DAYOFWEEK('2018-05-25');
-- 6

SELECT DAYOFYEAR('2018-05-25');
-- 145
```

- `DAYOFMONTH(date)` — день месяца;
- `DAYOFWEEK(date)` — номер дня недели;
- `DAYOFYEAR(date)` — номер дня в году.

```sql
SELECT MONTH('2018-05-25');
-- 5

SELECT YEAR('2018-05-25');
-- 2018

SELECT QUARTER('2018-05-25');
-- 2
```

- `MONTH()` — месяц;
- `YEAR()` — год;
- `QUARTER()` — квартал.

```sql
SELECT WEEK('2018-05-25', 1);
-- 21

SELECT LAST_DAY('2018-05-25');
-- 2018-05-31
```

- `WEEK(date, first)` — номер недели;
- `LAST_DAY(date)` — последний день месяца.

```sql
SELECT DAYNAME('2018-05-25');
-- Friday

SELECT MONTHNAME('2018-05-25');
-- May
```

- `DAYNAME()` — название дня недели;
- `MONTHNAME()` — название месяца.

---

### Извлечение частей времени

```sql
SELECT HOUR('21:25:54');
-- 21

SELECT MINUTE('21:25:54');
-- 25

SELECT SECOND('21:25:54');
-- 54
```

- `HOUR()` — часы;
- `MINUTE()` — минуты;
- `SECOND()` — секунды.

---

### EXTRACT

`EXTRACT()` — универсальная функция для извлечения части даты или времени.

```sql
EXTRACT(unit FROM datetime)
```

Примеры:

```sql
SELECT EXTRACT(YEAR FROM '2018-05-25 21:25:54');
-- 2018

SELECT EXTRACT(MONTH FROM '2018-05-25 21:25:54');
-- 5

SELECT EXTRACT(DAY FROM '2018-05-25 21:25:54');
-- 25

SELECT EXTRACT(HOUR FROM '2018-05-25 21:25:54');
-- 21

SELECT EXTRACT(MINUTE FROM '2018-05-25 21:25:54');
-- 25

SELECT EXTRACT(SECOND FROM '2018-05-25 21:25:54');
-- 54
```

Также можно извлекать составные части:

```sql
SELECT EXTRACT(YEAR_MONTH FROM '2018-05-25 21:25:54');
-- 201805

SELECT EXTRACT(HOUR_SECOND FROM '2018-05-25 21:25:54');
-- 212554
```

## 6.4. CASE, IF, IFNULL, COALESCE

Эти функции позволяют делать условную логику прямо внутри SQL-запроса.

---

### CASE

`CASE` проверяет несколько условий и возвращает результат для первого подходящего условия.

Общий вид:

```sql
CASE
    WHEN условие_1 THEN результат_1
    WHEN условие_2 THEN результат_2
    ELSE альтернативный_результат
END
```

Пример:

```sql
SELECT ProductName, ProductCount,
    CASE
        WHEN ProductCount = 1 THEN 'Товар заканчивается'
        WHEN ProductCount = 2 THEN 'Мало товара'
        WHEN ProductCount = 3 THEN 'Есть в наличии'
        ELSE 'Много товара'
    END AS Category
FROM Products;
```

Здесь каждому товару присваивается текстовая категория в зависимости от количества.

---

### IF

`IF()` — короткое условие из двух вариантов.

```sql
IF(условие, значение_если_истина, значение_если_ложь)
```

Пример:

```sql
SELECT ProductName, Manufacturer,
    IF(ProductCount > 3, 'Много товара', 'Мало товара') AS Category
FROM Products;
```

Если `ProductCount > 3`, вернётся `'Много товара'`, иначе — `'Мало товара'`.

---

### IFNULL

`IFNULL()` заменяет `NULL` на указанное значение.

```sql
IFNULL(выражение, значение_если_NULL)
```

Пример:

```sql
SELECT FirstName, LastName,
    IFNULL(Phone, 'не определено') AS Phone,
    IFNULL(Email, 'неизвестно') AS Email
FROM Clients;
```

Если `Phone` или `Email` равны `NULL`, вместо них выводится заданная строка.

---

### COALESCE

`COALESCE()` возвращает первое значение из списка, которое не равно `NULL`.

```sql
COALESCE(выражение_1, выражение_2, выражение_N)
```

Пример:

```sql
SELECT FirstName, LastName,
    COALESCE(Phone, Email, 'не определено') AS Contacts
FROM Clients;
```

Логика такая:

1. если есть `Phone`, выводится телефон;
2. если телефона нет, но есть `Email`, выводится email;
3. если нет ни телефона, ни email, выводится `'не определено'`.

## Мини-шпаргалка

### Строки

```sql
CONCAT(a, b)              -- объединить строки
CONCAT_WS(' ', a, b)      -- объединить через разделитель
LENGTH(str)               -- длина строки
LTRIM(str)                -- убрать пробелы слева
RTRIM(str)                -- убрать пробелы справа
TRIM(str)                 -- убрать пробелы с двух сторон
LOCATE(find, str)         -- найти позицию подстроки
LEFT(str, n)              -- взять n символов слева
RIGHT(str, n)             -- взять n символов справа
SUBSTRING(str, start, n)  -- взять подстроку
REPLACE(str, old, new)    -- заменить подстроку
LOWER(str)                -- нижний регистр
UPPER(str)                -- верхний регистр
REVERSE(str)              -- перевернуть строку
LPAD(str, len, pad)       -- дополнить слева
RPAD(str, len, pad)       -- дополнить справа
```

### Числа

```sql
ROUND(num, n)       -- округлить
TRUNCATE(num, n)    -- отбросить лишние знаки
ABS(num)            -- модуль
CEILING(num)        -- округлить вверх
FLOOR(num)          -- округлить вниз
POWER(num, degree)  -- степень
SQRT(num)           -- квадратный корень
SIGN(num)           -- знак числа
RAND()              -- случайное число от 0 до 1
```

### Даты и время

```sql
NOW()                  -- текущие дата и время
CURDATE()              -- текущая дата
CURTIME()              -- текущее время
DAYOFMONTH(date)       -- день месяца
DAYOFWEEK(date)        -- день недели
DAYOFYEAR(date)        -- день года
MONTH(date)            -- месяц
YEAR(date)             -- год
QUARTER(date)          -- квартал
WEEK(date, first)      -- номер недели
LAST_DAY(date)         -- последний день месяца
HOUR(time)             -- часы
MINUTE(time)           -- минуты
SECOND(time)           -- секунды
EXTRACT(unit FROM dt)  -- извлечь часть даты/времени
```

### Условия

```sql
CASE
    WHEN condition THEN result
    ELSE alternative
END

IF(condition, true_value, false_value)

IFNULL(value, replacement)

COALESCE(value1, value2, fallback)
```
