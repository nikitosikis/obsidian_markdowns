
# Введение в Matplotlib

Визуализация данных позволяет наглядно представить различные тредны, закономерности, сделать сухие данные более живыми, более доступными и понятными для аналитиков и простых пользователей. В экосистеме языка Python одним из наиболее популярных инструментов для визуализации данных является библиотека Matplotlib, которая позволяет создавать множество визуализаций, от простых линейных графиков до сложных 3D-графиков, с точным контролем над каждым аспектом визуализации. И в даном случае мы рассмотрим, как интегрировать функционал Pandas и Matplotlib в одном приложении и использовать функциональность Matplotlib для визуализации данных Pandas.

Для работы с Matplotlib нам естественно прежде всего надо установить эту библиотеку. Это можно с помощью менеджера пакетов pip с помощью команды:

```python
pip install matplotlib
```

После установки Matplotlib ее можно импортировать вместе с другими библиотеками в скрипт Python:

```python
import matplotlib.pyplot as plt
import pandas as pd
```

### Базовая структура графика Matplotlib

Каждый график Matplotlib, независимо от его сложности, имеет следующие основные компоненты:

- Данные. В основе каждого графика лежат точки данных, которые надо визуализировать. Эти точки данных могут быть представлены в различных формах, таких как списки, массивы или данные, загруженные из внешних источников. Тип имеющихся данных влияет на выбор графика и способ его настройки.
    
- Тип графика. Matplotlib предлагает множество типов графиков, включая линейные графики, столбчатые диаграммы, диаграммы рассеяния, гистограммы и другие. Выбор типа графика должен соответствовать характеру данных и цели визуализации.
    
- Настройки. Matplotlib предоставляет обширные возможности настройки, позволяя адаптировать график к конкретным потребностям. Можно изменять различные аспекты графика, такие как цвета, маркеры, стили линий, метки, заголовки, свойства осей и многое другое. Такой уровень настройки позволяет создавать визуально привлекательные и информативные визуализации.
    
- Отображение или сохранение. После создания графика его можно отобразить на экране или сохранить в файл для последующего использования или обмена. Для отображения графика используется функция `plt.show()`, а такие функции, как `plt.savefig()`, позволяют сохранять его в различных форматах (например, PNG и PDF).
    

Рассмотрим создание графика поэтапно.

#### Рисунок и оси

На верхнем уровне этой структуры находится рисунок или фигура (figure). Это пространство, контейнер, где будут отображаться графики и диаграммы.

На каждом рисунке есть одна или несколько осей. Ось (axis) в контексте Matplotlib представляет собой отдельный график или подграфик внутри фигуры. Оси предоставляют пространство, где можно визуализировать данные, устанавливать системы координат и применять различные настройки.

Для создания одного набора осей на графике можно использовать функцию subplots():

```python
# Создание графика с одним набором осей
figure, axes = plt.subplots()
print(figure)       # Figure(640x480)
print(axes)         # Axes(0.125,0.11;0.775x0.77)
```

Каждая ось в axes представляет собой отдельный набор осей на графике. Если нужно несколько подграфиков на одном графике, можно указать количество строк и столбцов, создав сетку осей:

```python
# Создание сетки подграфиков 2x2 на одном графике
figure, axes = plt.subplots(2, 2)
```

#### Метки

Для пояснения, что представляют оси, применяются метки. В зависимости от оси у объекта axes для установки меток применяются следующие методы:

- `set_xlabel()`: устанавливает метку для оси X, которая описывает измерение данных вдоль горизонтальной оси.
    
- `set_ylabel()`: устанавливает метку для оси Y, которая описывает измерение данных вдоль вертикальной оси.
    

Пример установки меток для осей:

```python
# Создание графика с одним набором осей
figure, axes = plt.subplots()
 
# устанавливаем метки для осей
axes.set_xlabel("День")
axes.set_ylabel("Цена")
```

Метки можно дополнительно настроить, указав такие атрибуты, как `fontsize` (размер шрифта), `fontfamily` (семейство шрифтов), `color` (цвет) и др. Например, вы можете изменить размер шрифта, чтобы сделать метки более заметными:

```python
axes.set_ylabel("Цена", fontsize=12)
```

#### Заголовок

Заголовок указывает на назначение графика. Это краткий, информативный текст, размещенный над графиком, чтобы передать цель или тему визуализации. Для добавления заголовка к графику у объекта axes применяется метод set_title():

```python
# Создание графика с одним набором осей
figure, axes = plt.subplots()
 
# устанавливаем метки для осей
axes.set_xlabel("День")
axes.set_ylabel("Цена")
 
# устанавливаем заголовок
axes.set_title("Динамика цены товара")
```

Заголовки также можно настроить в соответствии с предпочтитениями и улучшить их визуализацию. Здесь опять же можно изменить такие атрибуты, как `fontsize` (размер шрифта), `fontweight` (толщина шрифта), `fontfamily` (семейство шрифтов), `color` (цвет), `verticalalignment` (выравнивание по вертикали), `horizontalalignment` (выравнивание по горизонтали), `loc` (позиция). Например:

```python
axes.set_title("Динамика цены товара", fontsize=16, fontweight="bold", color="red", loc="center")
```

#### Деления осей

Деления представляют метки и подписи вдоль осей X и Y, которые указывают на определенные точки данных или интервалы. Для настройки делений у объекта axes применяются следующие методы:

- `set_xticks()`: задает положения делений по оси X
    
- `set_xticklabels()`: задает метки делений по оси X
    
- `set_yticks()`: задает положения делений по оси Y
    
- `set_yticklabels()`: задает метки делений по оси Y
    

Пример установки делений:

```python
# Создание графика с одним набором осей
figure, axes = plt.subplots()
 
# устанавливаем метки для осей
axes.set_xlabel("День")
axes.set_ylabel("Цена")
 
# устанавливаем заголовок
axes.set_title("Динамика цены товара", fontsize=16, fontweight="bold", color="red", loc="center")
 
# установка делений осей
axes.set_xticks([0, 1, 2, 3, 4])
axes.set_xticklabels(["Jan 1", "Feb 1", "Mar 1", "Apr 1", "May 1"])
axes.set_yticks([0, 10, 20, 30, 40])
axes.set_yticklabels(["500", "600", "700", "800", "900"])
```

Также можно настроить различные параметры делений, включая длину деления и метку деления и т.д.:

```python
# размер метки - 10, длина - 5
axes.tick_params(axis="x", length=5, labelsize=10)
```

#### Линии сетки

Линии сетки представляют горизонтальные и вертикальные линии, которые проходят по графику, помогая в интерпретации данных и выравнивании. Для добавления линий на график применяется метод `axes.grid()`. По умолчанию отображаются как основные, так и второстепенные линии сетки. Но можно настроить, какие линии сетки отображаются и как они выглядят, используя такие параметры, как `which`, `axis` и `linestyle`:

```python
Axes.grid(visible=None, which="major", axis="both", **kwargs)
```

Основные параметры функции:

- `visible`: указывает, отображать ли линии сетки (при значении `True`)
    
- `which`: линии сетки, к которым будут применяться изменения. Может принимать следующие значения: "major", "minor" и "both"
    
- `axis`: ось, к которой будут применяться изменения. Может принимать следующие значения: "both", "x" и "y"
    

Дополнительный параметр `kwargs` может устанавливать дополнительные параметры линий, такие как цвет, толщину, прозрачность и т.д. Например:

```python
axes.grid(True, which="major", axis="both", linestyle="--", color="gray", linewidth=0.5, alpha=0.7)
```

#### Легенда

Легенда позволяет уточнить, какие данные соответствуют каждой линии или маркеру. Чтобы создать легенду, надо через метод `Axes.plot` и его параметр `label` присвоить метки каждому отображаемому элементу, а затем вызвать `Axes.legend()`:

```python
axes.plot(x1, y1, label="Набор данных A")
axes.plot(x2, y2, label="Набор данных B")
axes.legend()
```

По умолчанию Matplotlib размещает легенды в условном "наилучшем" месте, чтобы избежать перекрытия с данными. Однако также можно указать местоположение легенды, используя параметр `loc` со значениями местоположения:

- `"best"` (лучшее расположение - выбирается автоматически, значение по умолчанию)
    
- `"upper right"` (вверху справа)
    
- `"upper left"` (вверху слева)
    
- `"lower left"` (внизу слева)
    
- `"lower right"` (внизу справа)
    
- `"right"` (справа)
    
- `"center left"` (в центре слева)
    
- `"center right"` (в центре справа)
    
- `"lower center"` (внизу по центру)
    
- `"upper center"` (вверху по центру)
    
- `"center"` (по центру)
    

Например:

```python
axes.legend(loc="upper right")
```

И также можно настроить внешний вид легенды, изменив такие свойства, как `fontsize` (размер шрифта), `fontweight` (толщина шрифта), `fontfamily` (семейство шрифтов) и др. Например:

```python
axes.legend(fontsize=12, framealpha=0.8)
```

#### Аннотации

Аннотации представляют текст или маркеры, добавляемые к определенным точкам данных или областям графика для предоставления дополнительной информации или выделения ключевых особенностей. Для добавления текстовых аннотаций применяется функция `axes.text()`, который принимает координаты текста, сам текст, а также может принимать ряд необязательных параметров:

```python
Axes.text(x, y, s, fontdict=None, **kwargs)
```

Например, текст "Макс. значение" с координатами (x=3, y = 16):

```python
axes.text(3, 16, "Макс. значение", fontsize=12, color="red", ha="center", va="center")
```

#### Отображение графика

После создания графика его можно отобразить на экране с помощью функции `plt.show()`:

```python

figure, axes = plt.subplots()
 
# устанавливаем метки для осей
axes.set_xlabel("День")

axes.set_xticks([0, 1, 2, 3, 4])
axes.set_xticklabels(["Jan 1", "Feb 1", "Mar 1", "Apr 1", "May 1"])
axes.set_yticks([0, 10, 20, 30, 40])
axes.set_yticklabels(["500", "600", "700", "800", "900"])
 
axes.grid()
 
plt.show()
```

# Интеграция Matplotlib и Pandas

### Построение графика

В прошлой теме мы рассмотрели различные отдельные компоненты создания графика с помощью библиотеки Matplotlib. Теперь определим все вместе и построим полноценный график:

```python
months = [1, 2, 3, 4, 5]
prices = [500, 630, 720, 800, 660]
 
# Создание графика с одним набором осей
figure, axes = plt.subplots()
 
# устанавливаем метки для осей
axes.set_xlabel("День")
axes.set_ylabel("Цена")
 
# Построение графика данных о температуре в виде линейного графика с настройками
axes.plot(months, prices, marker="o", linestyle="-", color="b", label="Цены по месяцам")
 
 
# устанавливаем заголовок
axes.set_title("Динамика цены товара", fontsize=16, fontweight="bold", color="red", loc="center")
 
# Настройка делений оси X
axes.set_xticks([1, 2, 3, 4, 5])
axes.set_xticklabels(["Jan 1", "Feb 1", "Mar 1", "Apr 1", "May 1"])
# Настройка делений оси Y
axes.set_yticks([500, 600, 700, 800])
axes.set_yticklabels(["500", "600", "700", "800"])
 
# Добавление линий сетки
axes.grid()
 
# Добавление легенды
axes.legend()
 
# Текстовая аннотация
axes.annotate("Максимальная\nцена (800)", xy=(4, 800), xytext=(4, 780), fontsize=10)
 
plt.show()
```

Здесь данные для отображения на графике представлены двумя списками:

```python
months = [1, 2, 3, 4, 5]
prices = [500, 630, 720, 800, 660]
```

Список month хранит номера месяцев, а prices - список соответствующих им цен для некоторого абстрактного товара.

Для построения графика передаем эти списки в метод `axes.plot()`

```python
axes.plot(months, prices, marker="o", linestyle="-", color="b", label="Цены по месяцам")
```

Где список months проецируется на ось X, а список prices - на ось Y. В итоге мы получим следующую визуализацию:

![[Pasted image 20260716210237.png]]

Хотя в примере выше в качестве источника данных для графика использовались простые списки Python, но равным образом мы могли бы использовать данные из структур Pandas - Series и DataFrame:

```python
months = pd.date_range(start="2025-01-01", end="2025-05-01", freq="MS")
prices = [500, 630, 720, 800, 660]
 
# Create a Pandas DataFrame
df = pd.DataFrame({"Month": months.strftime("%b %Y"), "Price": prices})
 
# Создание графика с одним набором осей
figure, axes = plt.subplots()
 
# устанавливаем метки для осей
axes.set_xlabel("Месяц")
axes.set_ylabel("Цена")
 
# Построение графика данных о температуре в виде линейного графика с настройками
axes.plot(df["Month"], df["Price"], marker="o", linestyle="-", color="b", label="Цены по месяцам")
 
# устанавливаем заголовок
axes.set_title("Динамика цены товара", fontsize=16, fontweight="bold", color="red", loc="center")
 
# Добавление линий сетки
axes.grid()
 
# Добавление легенды
axes.legend()
 
# Текстовая аннотация
axes.annotate("Максимальная\nцена (800)", xy=(3, 800), xytext=(3, 780), fontsize=10)
 
plt.show()
```

Здесь в качестве источника данных применяется набора данных df с двумя столбцами - "Month" и "Price". Первый столбец ("Month") применяется для оси X, а столбец "Price" - для столбца Y:

```python
axes.plot(df["Month"], df["Price"], marker="o", linestyle="-", color="b", label="Цены по месяцам")
```

Причем здесь мы не устанавливаем явным образом деления и метки на осях - они генерируются автоматически на основании данных в DataFrame. В целом мы получим график, аналогичный тому, что приведен в предыдущем примере.

### Столбчатая диаграмма

Линейные графики - единственный вид графиков/диаграм, который мы можем использовать в Matplotlib. Так, столбчатые диаграммы или гистограммы (Bar plots) представляют еще один полезный метод визуализации данных временных рядов, особенно при работе с категориальными или дискретными временными интервалами, такими как месяцы, годы или конкретные события.

На столбчатых диаграммах ось x обычно представляет категориальные или дискретные временные интервалы или события. Каждая категория или событие соответствует столбцу.

Ось y представляет числовые значения, связанные с каждой категорией или событием. Эти значения могут быть подсчетами, измерениями или любыми другими соответствующими данными.

Столбчатые диаграммы могут быть как вертикальными (по умолчанию), так и горизонтальными, в зависимости от выбранной ориентации. Рассмотрим небольшой пример:

```python
# Пример данных временного ряда 
months = pd.date_range(start="2025-01-01", end="2025-12-31", freq="MS")
 
# отображаемые значения - ежемесячные продажи
sales = [5500, 5800, 5900, 3500, 3500, 3800, 6100, 6500, 3900, 3600, 3300, 5800]
 
# Создание DataFrame Pandas
df = pd.DataFrame({"Month": months.strftime("%b %Y"), "Sales": sales})
 
# Создание вертикальной столбчатой ​​диаграммы для данных временного ряда
plt.figure(figsize=(12, 6))
 
plt.bar(df["Month"], df["Sales"], color="skyblue")
 
plt.xlabel("Месяц")
plt.ylabel("Продажи")
 
plt.title("Продажи по месяцам за год")
 
plt.xticks(rotation=45)     # Поворот меток оси x для лучшей читаемости
 
plt.grid()
 
# Отображение графика
plt.tight_layout()
plt.show()
```

Здесь аналогично по оси X у нас располагаются временные отрезки - месяца, и каждый столбик соответствует отдельному месяцу. А по оси Y - некоторые абстрактные данные по продажам. В целом здесь та же самая концепция, но в отличие от линейного графика, для отрисовки столбиков применяется метод `plt.bar()`, в который также передаются данные для проецирования на оси X и Y, и далее мы можем указать различные дополнительные необязательные параметры, как цвет столбиков:

```python
plt.bar(df["Month"], df["Sales"], color="skyblue")
```

### Диаграмма с областями

Рассмотрим еще один вид диаграмм, которые мы можем создавать с помощью Matplotlib - диаграмму с областями (area plot). Диграмма с областями представляет метод визуализации данных, используемый для представления данных временных рядов или данных с последовательными значениями во времени. Она похожа на линейный график, но отображает область между линией и осью X, заполненную цветом или узорами. Диаграммы с областями полезны для визуализации кумулятивных данных, показывая, как различные категории или компоненты вносят вклад в целое с течением времени.

Как правило, ось X на этой диаграмме представляет время или последовательные значения. Каждая точка на оси X соответствует определенному моменту времени или интервалу. А на ось Y также проецируются сами данные.

Рассмотрим создание диаграммы на небольшом примере:

```python
# помесячные данных временного ряда
months = pd.date_range(start="2025-01-01", end="2025-12-31", freq="MS")
 
# Списки для ежемесячных продаж для трех категорий товаров
laptops = [5500, 5800, 3000, 3500, 3500, 3800, 6100, 6500, 6600, 6600, 6800, 5000]
smartphones = [2500, 1600, 2700, 1800, 2900, 5000, 5100, 5500, 5300, 5600, 5500, 5600]

df = pd.DataFrame({"Month":months.strftime("%b %Y"), "Laptops": laptops, "Smartphones": smartphones, "Tablets": tablets})
```

Для создания самой диаграммы из DataFrame применяется метод `stackplot()`:

```python
plt.stackplot(df["Month"], df["Laptops"], df["Smartphones"], df["Tablets"], labels=["Ноутбуки", "Смартфоны", "Планшеты"])
```

В этот метод первым аргументом передается список данных для оси X (в нашем случае месяца). Затем передаются собственно данные, которые отображаются на диаграмме - наши данные по продажам трех типов товаров. То есть фактически мы передаем три списка данных, для каждого из которых будет создаваться своя область. И дополнительный параметр `labels` устанавливает текстовые метки для каждой области.
