# Создание и свойства DataFrame

DataFrame представляет двумерную структуру данных с метками. DataFrame состоит из строк и столбцов. DataFrame можно представить как электронную таблицу, таблицу SQL или словарь объектов Series. Как правило, это наиболее часто используемый объект pandas.

Особенности DataFrame:

- Это изменяемая, изменяемая по размеру и гетерогенная структура данных, где каждый столбец может иметь свой тип данных. Эта гибкость позволяет обрабатывать разнообразные наборы данных и выполнять широкий спектр операций над данными.
    
- DataFrame имеет оси с метками (строки и столбцы), что упрощает индексирование и обработку. Каждый столбец имеет уникальную метку, а каждая строка — уникальную метку строки (индекс).
    
- DataFrame построен на основе объектов Series, которые представляют собой одномерные массивы с метками. Каждый столбец в DataFrame представляет собой Series, имеющий общий индекс.
    
- API DataFrame предоставляет удобные методы и функции для очистки, преобразования, фильтрации и агрегирования данных. Эти операции могут эффективно выполняться с большими наборами данных. При чем Pandas оптимизирует использование памяти и предоставляет эффективные структуры данных, позволяя работать с большими наборами данных даже на машинах с ограниченными ресурсами.
    
- Объекты DataFrame легко интегрируются с другими популярными библиотеками Python, такими как NumPy, Matplotlib и scikit-learn, что делает их удобными для анализа данных, визуализации и задач машинного обучения.
    

### Создание DataFrame

Для создания DataFrame применяется следующая функция-конструктор:

```python
import pandas as pd
pd.DataFrame(data=None, index=none, columns=none, dtype=None, copy=None)
```

Параметры конструктора:

- `data`: собственно данные датафрейма - словари Python, объекты `Iterable`, массивы NumPy, объекты `Series` или другие объекты DataFrame
    
- `index`: индекс для использования в результирующем фрейме
    
- `columns`: метки столбцов для использования в результирующем фрейме, если в данных они отсутствуют, по умолчанию используется RangeIndex(0, 1, 2, …, n).
    
- `dtype`: тип данных (допускается только один тип данных).
    
- `copy`: логическое значение, которое указывает, надо ли копировать данные из входных данных
    

Первые три параметра (данные, индексы и столбцы) собственно и составляют три основных компонентов объекта DataFrame. В качестве данных DataFrame может принимать множество различных типов входных данных: словари Python, массивы NumPy, объекты `Series` или другие объекты DataFrame. Рассмотрим различные сценарии создания DataFrame.

#### Создание DataFrame из словаря

Например, создадим DataFrame из словаря Python:

```python
data = {
	'Name': ['Tom','Bob','Sam'],
	'Age': [41, 56, 31],
	'Company': ['Yandex','Vk','Sber']
}
df = pd.DataFrame(data)
>>> print(df)
  Name  Age   Company
0  Tom   41    Yandex
1  Bob   56        VK
2  Sam   31  Sberbank
```

Здесь мы создаем объект DataFrame из словаря Python, в котором ключи представляют собой имена столбцов ("Name", "Age", "Company"), а их значения - данные для каждого столбца. Таким образом, полученный DataFrame имеет три столбца.

#### Индексы

Индекс в DataFrame представляет метки, присвоенные каждой строке. Индекс помогает однозначно идентифицировать и получать доступ к конкретным строкам в DataFrame. По умолчанию Pandas присваивает числовой индекс, начиная с 0 (как видно из консольного вывода предыдущего примера). Но мы можем настроить индекс, используя собственные метки. Например:

```python
df = pd.DataFrame(data, index = ['1 person','2 person','3 person'])
>>> print(df)
         Name  Age   Company
1 person  Tom   41    Yandex
2 person  Bob   56        VK
3 person  Sam   31  Sberbank
```

#### Создание DataFrame из массива ndarray NumPy

Создание `DataFrame` из массива NumPy (объекта ndarray) представляет еще один способ построения DataFrame. Объект ndarray - это многомерный массив, который может содержать однородные данные. Каждый столбец в ndarray будет соответствовать столбцу в DataFrame. Рассмотрим следующий пример:

```python
import numpy as np
data = np.array([['Tom', 41, 'Yandex'],
	['Bob', 56, 'VK']
	['Sam', 31, 'Sberbank']])
df = pd.DataFrame(data)
>>> print(df)
     0   1         2
0  Tom  41    Yandex
1  Bob  56        VK
2  Sam  31  Sberbank
```

Здесь мы вначале создаем двумерный массив `ndarray` с именем data, который содержит три строки и три столбца.

По умолчанию при создании DataFrame каждому столбцу присваивается автосгенерированное имя на основе целочисленных индексов начиная с 0, как в примере выше. Но, возможно, мы захотим дать столбцам какие-то осмысленные названия. Это можно сделать с помощью параметра columns

Этот параметр представляет массив названий стобцов, где количество имен столбцов должно соответствовать количеству столбцов в массиве `ndarray`:

```python
columns = ['Name', 'Age', 'Company']
df = pd.DataFrame(data, index=columns)
>>> print(df)
  Name Age   Company
0  Tom  41    Yandex
1  Bob  56        VK
2  Sam  31  Sberbank
```

#### Форма DataFrame

Форма (shape) датафреймв представляет собой измерения - количество строк и столбцов в наборе данных. Поскольку DataFrame является двухмерной структурой, его форма состоит из двух измерений, где первой измерение представляет количество строк, а второе - столбцов. Для получения формы набора данных применяется атрибут shape:

```python
df = pd.DataFrame(data, index=columns)
>>> print(df.shape)
(3, 3) # 3 строки, 3 столбца
```

#### Тип данных

Структура данных `DataFrame` хранит данные тех же типов, что и объект `Series`. Для получения типа данных в DataFrame применяется атрибут `dtype`:

```python
>>> print(df.dtypes)
Name          object
Age           int64
Company       object
dtype: object
```

# Получение данных из DataFrame

Если данных очень много (сотни, тысячи строк), особенно, когда данные загружаются из внешних источников, то такой вывод на консоль может быть обременительным. Но подобно Series, Pandas также предоставляет для `DataFrame` методы для быстрого предварительного просмотра данных из `DataFrame`: `head(), tail()` и `sample()`. Эти методы позволяют получить общее представление о структуре DataFrame, именах столбцов и подмножестве данных. Хотя эти методы похожи на свои аналоги в Series, они работают со всем DataFrame, а не с одним столбцом.

### head()

Метод head() позволяет просмотреть первые несколько строк DataFrame. По умолчанию он отображает первые пять строк. Параметр n позволяет задать точное количество просматриваемых строк

```python
import pandas as pd

data = {

"Город": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Казань", "Красноярск", "Нижний Новгород", "Челябинск", "Уфа", "Краснодар", "Самара", "Ростов-на-Дону", "Омск", "Воронеж", "Пермь", "Волгоград"],

"Население": [13274, 5653, 1637, 1548, 1330, 1212, 1198, 1177, 1166, 1155, 1154, 1143, 1101, 1042, 1028, 1012]

}

df = pd.DataFrame(data)

print(df.head()) # первые 5

print()

print(df.head(n=3)) # первые 3

# или просто df.head(3)
             Город  Население
0           Москва      13274
1  Санкт-Петербург       5653
2      Новосибирск       1637
3     Екатеринбург       1548
4           Казань       1330

             Город  Население
0           Москва      13274
1  Санкт-Петербург       5653
2      Новосибирск       1637
```

### tail()

Метод tail() извлекает последние несколько строк DataFrame. По умолчанию он возвращает последние пять строк, но также в качестве аргумента можно передать количество отображаемых строк:

```python
import pandas as pd

data = {

"Город": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Казань", "Красноярск", "Нижний Новгород", "Челябинск", "Уфа", "Краснодар", "Самара", "Ростов-на-Дону", "Омск", "Воронеж", "Пермь", "Волгоград"],

"Население": [13274, 5653, 1637, 1548, 1330, 1212, 1198, 1177, 1166, 1155, 1154, 1143, 1101, 1042, 1028, 1012]

}

df = pd.DataFrame(data)

print(df.tail()) # последние 5

print()

print(df.tail(3)) # последние 3

             Город  Население
11  Ростов-на-Дону       1143
12            Омск       1101
13         Воронеж       1042
14           Пермь       1028
15       Волгоград       1012

        Город  Население
13    Воронеж       1042
14      Пермь       1028
15  Волгоград       1012
```

### sample()

Метод sample() извлекает случайное подмножество строк из DataFrame. Это позволяет получить представление о случайных записях и понять общее распределение данных. Синтаксис использования `sample()` выглядит следующим образом:

```python
df.sample(n=5, random_state=None)
```

Параметр `n` указывает на количество строк для извлечения, а `random_state` - необязательный параметр для установки конкретного случайного состояния для воспроизводимости:

```python
import pandas as pd

data = {

"Город": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Казань", "Красноярск", "Нижний Новгород", "Челябинск", "Уфа", "Краснодар", "Самара", "Ростов-на-Дону", "Омск", "Воронеж", "Пермь", "Волгоград"],

"Население": [13274, 5653, 1637, 1548, 1330, 1212, 1198, 1177, 1166, 1155, 1154, 1143, 1101, 1042, 1028, 1012]

}

df = pd.DataFrame(data)

print(df.sample(4)) # случайные 4 строки

              Город  Население
13          Воронеж       1042
0            Москва      13274
1   Санкт-Петербург       5653
5        Красноярск       1212
```

Параметр `random_state` можно использовать для установки конкретного начального значения генератора случайных чисел, что обеспечивает воспроизводимость результатов при использовании одного и того же состояния генератора случайных чисел. При этом случайные числа генерируются не по порядку, соответственно выбираемые строки то же будут идти не по порядку.

# Выбор столбцов и строк

Pandas позволяет на основе желаемых критериев (имена столбцов, метки строк или определенные условия) выбирать произвольные столбцы и строки из `DataFrame`. Это позволяет нам сосредоточиться на релевантных данных для их анализа, визуализации и моделирования.

### Выбор столбцов

Каждый отдельный столбец в DataFrame представляет объект `Series`. Pandas предоставляет несколько способов выбора столбцов. Прежде всего это можно сделать с помощью квадратных скобок, передав в них имя столбца, например:

```python
import pandas as pd

data = {

"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Казань", "Красноярск", "Нижний Новгород", "Волгоград"],

"Population": [13274, 5653, 1637, 1548, 1330, 1212, 1198, 1012],

"Time zone": ["GMT+3", "GMT+3","GMT+7","GMT+5","GMT+3","GMT+7","GMT+3","GMT+3"]

}

df = pd.DataFrame(data)

# получаем серию для столбца City

cities = df["City"]

print(cities)

0             Москва
1    Санкт-Петербург
2        Новосибирск
3       Екатеринбург
4             Казань
5         Красноярск
6    Нижний Новгород
7          Волгоград
Name: City, dtype: object
```

Выражение `df["City"]` возвращает объект Series, который представляет собой одномерный массив с метками, содержащий значения из столбца "City".

Другой способ получения столбца представляет точечная нотация (dot notation): после имени объекта DataFrame следует точка и имя столбца, например, `df.column_name`:

```python
import pandas as pd

data = {

"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Казань", "Красноярск", "Нижний Новгород", "Волгоград"],

"Population": [13274, 5653, 1637, 1548, 1330, 1212, 1198, 1012],

"Time zone": ["GMT+3", "GMT+3","GMT+7","GMT+5","GMT+3","GMT+7","GMT+3","GMT+3"]

}

df = pd.DataFrame(data)

# получаем серию для столбца City

cities = df.City

print(cities)
```

#### Выбор нескольких столбцов

Чтобы выбрать несколько столбцов, в квадратных скобках указывается список имен столбцов:

```python
columns = ['City', 'Time Zone']
cities = df[columns]
>>> print(cities)

              City Time zone
0           Москва     GMT+3
1  Санкт-Петербург     GMT+3
2      Новосибирск     GMT+7
3     Екатеринбург     GMT+5
4           Казань     GMT+3
5       Красноярск     GMT+7
6  Нижний Новгород     GMT+3
7        Волгоград     GMT+3
```

#### Выбор столбцов по типу данных

Иногда полезно выбирать столбцы на основе их типов данных. Pandas предоставляет метод select_dtypes() для этого. Например, чтобы выбрать из DataFrame только числовые столбцы, можно использовать следующий код:

```python
df.select_dtypes(include='number')
```

К примеру, выбирем столбцы, у которых тип данных int64:

```python
int_columns = df.select_dtypes(include='int64')
>>> print(int_columns)
   Population
0       13274
1        5653
2        1637
3        1548
4        1330
5        1212
6        1198
7        1012
```

### Выбор строк в DataFrame

Аналогично выбору столбцов из DataFrame, существует несколько способов выбора строк.

#### Выбор строк с помощью срезов

С помощью стандартного синтаксиса срезов Python можно выбрать подмножество строк из DataFrame. При срезе DataFrame указываются индексы начальной и конечной точек среза. Полученный срез включает начальный индекс и исключает конечный индекс, следуя стандартному соглашению Python о срезах списков. Например:

```python
df_slice = df[1:4]
>>> print(df_slice)
              City  Population Time zone
1  Санкт-Петербург        5653     GMT+3
2      Новосибирск        1637     GMT+7
3     Екатеринбург        1548     GMT+5
```

Если необходимо выбрать все строки от начала до определенного индекса, можно опустить первый индекс. Например, чтобы выбрать все строки до (но не включая) индекса 5, можно использовать:

```python
df[:5]
```

Аналогично, если надо выбрать с определенного индекса до самого конца, опускается второй индекс. Например выбора с 5 индекса:

```python
df[5:]
```

### Выбор столбца строки

Комбинируя два индекса - по стообцу и строке, мы можем выбрать определенные ячейки, которые расположены на пересечении определенных строк и столбцов. Например:

```python
df_select = df['City'][2]
>>> print(df_select)
Новосибирск
```

В данном случае выражение `df["City"][2]` выбирает значение ячейку из столбца "City" в 3-й строке (с индексом 2).

Аналогично мы можем выбрать несколько строк из определенного столбца. Например, выберем 3 и 4 строки из столбца City:

```python
df_select = df['City'][2:4]
>>> print(df_select)
2     Новосибирск
3    Екатеринбург
Name: City, dtype: object
```

Но синтаксис срезов подходит, если строки идут по порядку. А если не по порядку, то можно передать список с индексами выбираемых строк:

```python
df_select = df['City'][[0,2,4]]
>>> print(df_select)
0         Москва
2    Новосибирск
4         Казань
Name: City, dtype: object
```

И также в случае, если нам надо выбрать несколько столбцов, мы можем передать список выбираемых столбцов:

```python
columns = ['City', 'Population']
df_select = df[columns][1:4]
>>> print(df_select)
              City  Population
1  Санкт-Петербург        5653
2      Новосибирск        1637
3     Екатеринбург        1548
```

# Выбор данных с помощью методов loc и iloc

Методы loc и iloc упрощают выбор данных из DataFrame.

### Метод loc

Метод loc для выбора данных применяет метки строк или столбцов, которые мы хотим выбрать. Синтаксис:

```python
df.loc[метка_строки, метка_столбца]
```

Например, выберем 2 строку:

```python
import pandas as pd

data = {

"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Казань", "Красноярск", "Нижний Новгород", "Челябинск", "Уфа", "Краснодар", "Самара", "Ростов-на-Дону", "Омск", "Воронеж", "Пермь", "Волгоград"],

"Population": [13274, 5653, 1637, 1548, 1330, 1212, 1198, 1177, 1166, 1155, 1154, 1143, 1101, 1042, 1028, 1012]

}

df = pd.DataFrame(data)

# выбираем вторую строку (индекс 1, так как индексация начинается с 0)

single_row = df.loc[1]

>>> print(single_row)

City          Санкт-Петербург
Population               5653
Name: 1, dtype: object
```

По умолчанию, если мы вручную не присваиваем метки строкам, то им атоматически присваиваются метки на основе их индексов. То есть первая строка будет иметь индекс/метку 0, вторая строка - 1 и так далее. Здесь, чтобы получить вторую строку, мы используем выражение `df.loc[1]`.

Если нам надо получить подряд несколько строк, то можно использовать синтаксис срезов. Например, получим первые три строки:

```python
rows = df.loc[0:2]
>>> print(rows)
              City  Population
0           Москва       13274
1  Санкт-Петербург        5653
2      Новосибирск        1637
```

Выражение `df.loc[0:2]` указывает, что нам надо получить строки с 0 до 2 индекса.

Поскольку в данном случае выборка шла с самого начала, то мы могли бы опустить начальную метку:

```python
rows = df.loc[0:2]
```

Аналогично, если мы хотим выбрать данные до конца по самую последнюю строку включительно, то опускается вторая метка. Например, выберем с 10-й строки до конца:

```python
rows = df.loc[9:]
```

Если же нам надо выбрать несколько строк, но не подряд, например, строки с индексами 1, 2, 7, 9, 10, то тогда методу `loc()` можно передать список меток строк для выбора:

```python
row_labels = [1, 2, 7, 9, 10]
rows = df.loc[row_labels]
>>> print(rows)
               City  Population
1   Санкт-Петербург        5653
2       Новосибирск        1637
7         Челябинск        1177
9         Краснодар        1155
10           Самара        1154
```

Но метки строк также могут устанавливаться вручную. Рассмотрим подобную ситуацию:

```python
import pandas as pd

data = {

"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],

"Population": [13274, 5653, 1637, 1548, 1012]

}

# вручную определяем метки

row_labels = ["city1", "city2", "city3", "city4", "city5"]

# устанавливаем метки для строк

df = pd.DataFrame(data, index = row_labels)

# получаем строку с меткой city3

city3 = df.loc["city3"]

>>> print(city3)
City          Новосибирск
Population           1637
Name: city3, dtype: object
```

#### Выбор столбцов

Второй параметр метода `loc()` представляет метки выбираемых столбцов. Например, выберем столбец с именем "City":

```python
cities = df.loc[:, 'City']
>>> print(cities)
0             Москва
1    Санкт-Петербург
2        Новосибирск
3       Екатеринбург
4          Волгоград
Name: City, dtype: object
```

Здесь обратите внимание, что первый параметр извлекает все строки - от начала до конца, так как начальный и конечный индексы опущены

### Метод iloc

Метод iloc() применяется для выбора строк и столбцов по номерам в порядке их расположения в DataFrame, то есть по сути для выбора используются целочисленные индексы (а не метки как в `loc()`). Как обычно, индексация начинается с 0, поэтому индекс первого столбца/первой строки - 0. Синтаксис метода:

```python
df.iloc[индекс_строки, индекс_столбца]
```

#### Выбор строк

Для выбора одной строки из DataFrame в `iloc()` передается ее числовой индекса:

```python
single_row = df.iloc[1]
>>> print(single_row)
City          Санкт-Петербург
Population               5653
Name: 1, dtype: object
```

#### Выбор нескольких строк

Для множественного выбора строк подряд можно использовать синтаксис срезов:

```python
df.iloc[start_index:end_index]
```

Где `start_index` - индекс первой строки (включительно), `end_index` - индекс последней строки, до которой надо произвести выборку (НЕ включая). Например, выберем со второй строки (включая) по четвертую строку (не включая):

```python
rows = df.iloc[1:3]
>>> print(rows)
              City  Population
1  Санкт-Петербург        5653
2      Новосибирск        1637
```

Если надо выбрать строки, которые располагаются не по порядку, то первому аргументу метода передается список индексов выбираемых строк:

```python
rows = df.iloc[[1,3]]
>>> print(rows)
              City  Population
1  Санкт-Петербург        5653
3     Екатеринбург        1548
```

#### Выбор столбцов

Второй аргумент метода iloc() получает индексы выбираемых столбцов. И здесь работает та же логика, что и при выборе строк. Так, если мы хотим получить только один столбец, то передаем только один индекс. Например, выберем первый столбец:

```python
cities = df.iloc[:, 0]
>>> print(cities)
0             Москва
1    Санкт-Петербург
2        Новосибирск
3       Екатеринбург
4          Волгоград
Name: City, dtype: object
```

При выборе столбца нам все равно надо указать выбираемые строки через первый параметр. В примере выше `df.iloc[:, 0]` с помощью выражения `:` выбираем все строки.

И как и в случае со строками, если надо выбрать несколько столбцов, то можно передать список индексов выбираемых столбцов. Например, выбор первых двух столбцов:

```python
cities = df.iloc[:, [0,1]]
```

Если столбцы, которые надо выбрать, идут подряд, то можно использовать синтаксис срезов. Например, выбор до столбца с индексом 1:

```python
cities = df.iloc[:, :1]
```

# Загрузка данных из файлов и веб-страниц

Нередко данные поставляются в виде отдельных файлов. Pandas предоставляет несколько функций и методов, которые позволяют считать данные из файлов в различных форматах, таких как CSV, Excel, JSON, HTML и SQL или, наборот, записывать данные в файлы. Рассмотрим работу с расспространенными форматами.

### Загрузка данных из CSV-файлов

Одним из распространенных форматов файлов для хранения табличных данных является формат CSV (Comma-S Values). CSV-файл состоит из строк и столбцов, разделенных запятыми или другими разделителями. Pandas предоставляет средства для чтения данных из CSV-файлов в объект DataFrame с помощью функции pandas.read_csv(). Эта функция принимает больше 40 параметров, поэтому рассмотрим основные. Прежде всего в качестве первого и единственного обязательного аргумента необходимо передать путь к файлу или файлоподобный объект.

Допустим любой строковый путь. Строка может быть URL-адресом. Допустимые схемы URL включают http, ftp, s3, gs и file. Для файловых URL-адресов ожидается указание хоста. Локальный файл может выглядеть так: `file://localhost/path/to/table.csv`.

Если вы хотите передать объект пути, pandas принимает любой `os.PathLike`. Под файловоподобным объектом подразумеваются объекты с методом `read()`, такие как дескриптор файла (например, открываемый через встроенную функцию `open()`) или `StringIO`.

#### Дополнительные параметры

Также можно указать другие необязательные аргументы для настройки поведения функции. Некоторые из наиболее часто используемых параметров:

- `sep`: указывает, какой разделитель используется в CSV-файле. Значение по умолчанию - ",".
    
- `header`: указывает, какая строка CSV-файла должна использоваться в качестве имен столбцов. Значение по умолчанию - 0.
    
- `index_col`: указывает, какой столбец CSV-файла должен использоваться в качестве индекса. Значение по умолчанию - `None`.
    
- `usecols`: указывает, какие столбцы CSV-файла должны быть считаны в DataFrame. Значение по умолчанию - `None`, что означает, что будут считаны все столбцы.
    
- `dtype`: указывает тип данных каждого столбца в DataFrame. Значение по умолчанию - `None`, что означает, что pandas попытается определить типы данных автоматически.
    
- `skiprows`: указывает, сколько строк следует пропустить с начала CSV-файла. Значение по умолчанию - 0.
    
- `nrows`: определяет, сколько строк следует прочитать из CSV-файла. Значение по умолчанию - `None`, что означает чтение всех строк.
    

К примеру, используем параметр index_col. Этот параметр позволяет использовать определенный столбец в качестве индекса. Так, в нашем примере с рейтингом языков программирования Pandas по умолчанию добавляет дополнительный столбец с индексами начиная с 0. Но в нашем случае мы могли бы использовать первый столбец, который показывает место каждого языка (оно же уникально). Тогда параметру надо передать название данного столбца:

```python
df = pd.read_csv('langs.csv', index_col='Jan 2026')
>>> print(df.head())
          Jan 2025 Language  Ratings   Change
Jan 2026                                     
1                1   Python   22.61%   -0.68%
2                4        C   10.99%   +2.13%
3                3     Java    8.71%   -1.44%
4                2      C++    8.67%   -1.62%
5                5       C#    7.39%   +2.94%
```

Если мы хотим прочитать не все, а только определенные столбцы из CSV-файла, мы можем использовать параметр usecols и передать ему список имен столбцов или их номеров. Например, если мы хотим прочитать только столбцы с местом языка и само название языка (то есть столбцы "Jan 2026" и "Language"), то мы можем написать следующим образом:

```python
df = pd.read_csv('langs.csv', usecols=['Jan 2026', 'Language'])
>>> print(df.head())
   Jan 2026 Language
0         1   Python
1         2        C
2         3     Java
3         4      C++
4         5       C#
```

Вместо названий столбцов мы можем передать их индексы (особенно, если названия содержат много текста):

```python
df = pd.read_csv('langs.csv', usecols=[0,2,3])
```

Если мы хотим пропустить несколько строк с начала CSV-файла, мы можем передать параметру skiprows количество строк, которые надо пропустить. Например, если мы хотим пропустить первые две строки, мы можем передать `skiprows=2`:

```python
df = pd.read_csv('langs.csv', sciprows=2)
```

Если, наоборот, мы хотим прочитать только определенное количество строк из CSV-файла, мы можем использовать параметра nrows, которому передается количество считываемых строк. Например, если мы хотим прочитать только первые две строки, то мы можем передать `nrows=2`:

```python
df = pd.read_csv('langs.csv', nrows=2)
```

Параметр header может быть полезен, если мы хотим использовать другие названия столбцов. В этом случае мы можем передать параметру header номер строки, которую мы хотим использовать. Например, используем вторую строку (индексация начинается с 0):

```python
df = pd.read_csv('langs.csv', header=1)
```

## Загрузка данных из файлов Excel

Excel на сегодняшний день остается популярным приложением для работы с электронными таблицами, которое позволяет хранить и обрабатывать табличные данные. Файлы Excel имеют расширение .xls и .xlsx и могут содержать несколько листов или вкладок. Для их чтения в объект DataFrame Pandas предоставляет функцию pandas.read_excel(). Пример использования pandas.read_excel() для чтения файла Excel в объект `DataFrame`:

df = pd.read_excel('people.xlsx')

df = pd.read_excel('Excek.xlsx', sheet_name='langs')

# Сохранение данных в файлы

Сохранение данных из объекта `DataFrame` в Pandas представляет одну из самых частых задач в анализе данных. И для этого библиотека Pandas предлагает встроенные методы для большинства популярных форматов. Вкратце рассмотрим основные способы экспорта.

### CSV (Comma-Separated Values)

**CSV** представляет самый универсальный формат. Для сохранения в csv-файл Pandas предоставляет функцию to_csv(), где первый параметр представляет путь к файлу для сохранения. Распространенные дополнительные параметры:

- `index`: по умолчанию Pandas добавляет колонку с индексами, которую часто отключают с помощью параметра `index=False`.
    
- `sep`: разделитель разделитель в выходном файле, например, `sep=";"`
    

Пример сохранения в csv:

```python
import pandas as pd

data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Казань", "Красноярск", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1330, 1212, 1012],
"Time zone": ["GMT+3", "GMT+3","GMT+7","GMT+5","GMT+3","GMT+7","GMT+3"]
}

df = pd.DataFrame(data)
# Получаем города, где население меньше 1500 тыс. чел.

```

# Добавление и удаление строк и столбцов в DataFrame

### Добавление и удаление столбцов

#### Добавление столбцов

Добавление столбцов в DataFrame позволяет включать дополнительную информацию или вычисляемые признаки на основе существующих данных. Чтобы добавить один столбец, мы можем использовать оператор присваивания (=) для присвоения новой серии или массива метке столбца в DataFrame. Размер новых данных должна совпадать с длиной текущего DataFrame. Например:

```python
import pandas as pd
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012]
}

df = pd.DataFrame(data)
years = [1147, 1703, 1723, 1893, 1589]
# Добавление одного столбца
df["Year"] = years
>>> print(df)
              City  Population  Year
0           Москва       13274  1147
1  Санкт-Петербург        5653  1703
2      Новосибирск        1637  1723
3     Екатеринбург        1548  1893
4        Волгоград        1012  1589
```

Здесь в DataFrame добавляем столбец "Year" (условно год основания городов). Добавление столбца довольно просто: в квадратных скобках ([ ]) указываем метку нового столбца ("Year") и присваиваем список со значениями для этого столбца. 
#### Добавление нескольких столбцов

Чтобы добавить несколько столбцов одновременно, мы можем использовать функцию pd.concat() для объединения текущего DataFrame с новыми столбцами. Фактически это выглядит как объединение двух DataFrame, один из которых представляет новые столбцы:

```python
import pandas as pd
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012]
}

df = pd.DataFrame(data)
new_columns = pd.DataFrame({
"Year": [1147, 1703, 1723, 1893, 1589],
"Time zone": ["GMT+3", "GMT+3","GMT+7","GMT+5","GMT+3"]
})

df = pd.concat([df, new_columns], axis=1)
>>> print(df)
              City  Population  Year Time zone
0           Москва       13274  1147     GMT+3
1  Санкт-Петербург        5653  1703     GMT+3
2      Новосибирск        1637  1723     GMT+7
3     Екатеринбург        1548  1893     GMT+5
4        Волгоград        1012  1589     GMT+3
```

Здесь мы создаем новый DataFrame с именем new_columns, используя конструктор pd.DataFrame(). Этот набор данных содержит два столбца: "Year" и "Time zone".

Затем используем функцию `pd.concat()` для объединения существующего DataFrame df с новым DataFrame new_columns. Указав `axis=1`, мы гарантируем, что объединение будет выполняться по столбцам.

После выполнения кода мы присваиваем объединённый DataFrame обратно в df. В результате будет получен DataFrame, включающий исходные столбцы ("City" и "Population"), а также добавленные новые столбцы ("Year" и "Time zone").

#### Удаление столбцов

Удаление ненужных или избыточных столбцов из DataFrame может повысить ясность данных и эффективность анализа. Чтобы удалить один столбец, мы можем использовать метод drop(). В этот метод передается метка столбца и дополнительный параметр `axis=1`, который указывает, что мы удаляем столбец:

```python
df.drop('метка_столбца', axis=1, inplace=True)
```

Установив `inplace=True` через третий параметр, мы изменяем DataFrame напрямую, то есть изменения применяются к самому DataFrame, а не возвращают новый DataFrame.

Например, удалим столбец "Year":

```python
df = df.drop('Year', axis=1)
>>> print(df)
              City  Population Time zone
0           Москва       13274     GMT+3
1  Санкт-Петербург        5653     GMT+3
2      Новосибирск        1637     GMT+7
3     Екатеринбург        1548     GMT+5
4        Волгоград        1012     GMT+3
```

Если надо удалить несколько столбцов, то их метки передаются в виде списка первому параметру метода `drop()`:

```python
df = df.drop(['Year', 'Population'], axis=1)
```

### Добавление и удаление строк

#### Добавление одной строки

Чтобы добавить одну строку, обычно создается новая строка в виде объекта Series или словаря, которые содержат значения для каждого столбца. Затем новая строка присваивается определенному индексу в DataFrame с помощью метода loc(). Например:

```python
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012]
}

df = pd.DataFrame(data)
new_row = pd.Series(["Воронеж", 1042], index=df.columns)
df.loc[len(df)] = new_row
>>> print(df)
              City  Population
0           Москва       13274
1  Санкт-Петербург        5653
2      Новосибирск        1637
3     Екатеринбург        1548
4        Волгоград        1012
5          Воронеж        1042
```

Здесь создается новая строка в виде объекта `Series` с именем new_row. Значения `["Воронеж", 1042]` представляют данные для новой строки, а параметр `index=df.columns` используется для установки индекса новой строки в соответствии с именами столбцов в наборе данных df.

Выражение `len(df)` выступает в качестве метки индекса для указания местоположения новой строки. Присваивая по индексу `df.loc[len(df)]` значение new_row, тем самым добавляем новую строку в df.

#### Добавление нескольких строк

Чтобы добавить несколько строк в DataFrame, можно использовать метод concat() для объединения существующего объекта DataFrame с новым DataFrame, который содержит дополнительные строки. Например:

```python
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012]
}

df = pd.DataFrame(data)
new_rows = pd.DataFrame([["Воронеж", 1042], ['Казань', 1330]], columns=df.columns)
df = pd.concat([df, new_rows], ignore_index=True)
>>> print(df)
              City  Population
0           Москва       13274
1  Санкт-Петербург        5653
2      Новосибирск        1637
3     Екатеринбург        1548
4        Волгоград        1012
5          Воронеж        1042
6           Казань        1330
```

Здесь новый DataFrame с именем new_rows создается с помощью конструктора pd.DataFrame(). Он состоит из списка списков, где каждый внутренний список представляет собой строку. Имена столбцов для нового DataFrame указываются как `df.columns` для обеспечения выравнивания с существующим DataFrame df. Каждый внутренний список содержит значения для соответствующих столбцов, представляющие данные для каждой добавляемой строки.

Функция `pd.concat()` используется для объединения DataFrame с new_rows, фактически добавляя новые строки к существующему DataFrame. Установка `ignore_index=True` генерирует новый индекс для результирующего DataFrame, обеспечивая непрерывный индекс без дубликатов.
### Удаление строк

#### Удаление одной строки

Удаление одной строки из DataFrame можно выполнить с помощью метода drop(). Этот метод позволяет указать индекс или метку строки, которую надо удалить:

```python
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012]
}

df = pd.DataFrame(data)
df = df.drop(index=2)
>>> print(df)
              City  Population
0           Москва       13274
1  Санкт-Петербург        5653
3     Екатеринбург        1548
4        Волгоград        1012
```

С помощью параметра `index=2` указываем, что надо удалить строку с индексом 2 (третью строку).
#### Удаление нескольких строк

Чтобы удалить несколько строк из DataFrame, в метод `drop()` передается список меток индексов или меток удаляемых строк:

```python
rows = [1,3]
df = df.drop(index=rows)
>>> print(df)
          City  Population
0       Москва       13274
2  Новосибирск        1637
4    Волгоград        1012
```
Для удаления нескольких строк здесь создаем список rows, который содержит индексы удаляемых строк. В данном случае удаляем строки с индексами 1 и 3.

# Переименование и перестановка столбцов

### Переименование столбцов

Нередко данные загружаются из внешних источников, например, файла csv. И в этом случае столбцы в DataFrame получают те имена, которые определены в этом внешнем источники. Переименование столбцов в DataFrame позволяет изменять имена столбцов для повышения ясности, согласованности или приведения их в соответствие с конкретными требованиями. Переименование столбцов позволяет нам присваивать четкие и описательные имена, которые точно представляют содержащиеся в них данные. Четкие имена столбцов обеспечивают эффективную коммуникацию и сотрудничество между аналитиками данных, гарантируя, что все будут интерпретировать данные согласованно. Переименование столбцов помогает установить единые правила именования во всем DataFrame. Придерживаясь стандартизированной схемы именования, становится проще организовывать и сравнивать данные в разных наборах данных или столбцах. Согласованные имена столбцов также облегчают интеграцию и слияние данных, особенно при объединении нескольких DataFrame или выполнении операций с базой данных. Кроме того, переименование столбцов может значительно повысить эффективность анализа данных. Интуитивно понятные и информативные имена столбцов упрощают исследование и обработку данных. Становится проще выбирать, фильтровать, агрегировать или выполнять вычисления по конкретным столбцам, улучшая общий аналитический рабочий процесс. Хорошо названные столбцы экономят время, устраняя необходимость в избыточной документации или дополнительных этапах предварительной обработки данных.

Переименование отдельного столбца в DataFrame выполняется с помощью метода rename():

```python
df.rename(columns={'старое_имя_столбца': 'новое_имя_столбца'}, inplace=True)
```

В качестве первого параметра - параметра columns передается объект, который похож на словарь и который сопоставляет текущее имя столбца с новым именем. А значение `inplace=True` изменяет DataFrame напрямую, гарантируя, что изменения применяются к самому DataFrame. Например, переименуем столбец с "City" на "Name":

```python
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012]
}

df = pd.DataFrame(data)
df = df.rename(columns={"City": "Name"})
>>> print(df)
              Name  Population
0           Москва       13274
1  Санкт-Петербург        5653
2      Новосибирск        1637
3     Екатеринбург        1548
4        Волгоград        1012
```

Для переименования одновременно нескольких столбцов в методе rename() первому параметру передается словарь с парами `"старое_название":"новое_название"`. Например:

```python
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012]
}

df = pd.DataFrame(data)
df.rename(columns={"City": "Название", "Population": "Население"}, inplace=True)
>>> print(df)
          Название  Население
0           Москва      13274
1  Санкт-Петербург       5653
2      Новосибирск       1637
3     Екатеринбург       1548
4        Волгоград       1012
```

### Изменение порядка столбцов

Изменение порядка столбцов в DataFrame позволяет переставить столбцы для лучшего соответствия потребностям анализа, повышения читаемости или выполнения конкретных требований. В Pandas есть несколько подходов к изменению порядка столбцов в DataFrame.

#### Изменение порядка с помощью индексации

Изменение порядка столбцов с помощью индексации означает изменение последовательности или порядка столбцов в DataFrame путем явного выбора и упорядочивания их на основе их меток. Это обеспечивает простой и прямой подход к перестановке столбцов в соответствии с конкретными требованиями.

При данном подходе необходимо создать новый DataFrame, проиндексировав существующий DataFrame списком имен столбцов в желаемом порядке. Рассмотрим простой пример:

```python
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012],
"Time zone": ["GMT+3", "GMT+3","GMT+7","GMT+5","GMT+3"]
}

df = pd.DataFrame(data)
df = df[["City", "Time zone", "Population"]]
>>>print(df)
              City Time zone  Population
0           Москва     GMT+3       13274
1  Санкт-Петербург     GMT+3        5653
2      Новосибирск     GMT+7        1637
3     Екатеринбург     GMT+5        1548
4        Волгоград     GMT+3        1012
```

Для перестановки порядка столбцов передавем список их имен в квадратных скобках:

```python
df[["City", "Time zone", "Population"]]
```

В итоге столбцы будут идти в новом наборе DataFrame в том порядке, в котором они перечислены в этом списке.
#### Изменение порядка столбцов с помощью метода reindex()

Метод reindex() возвращает новый DataFrame с указанным порядком столбцов. Для изменения порядка столбцов через параметр columns метода `reindex()` передается новый список имен столбцов:

```python
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012],
"Time zone": ["GMT+3", "GMT+3","GMT+7","GMT+5","GMT+3"]
}

df = pd.DataFrame(data)

columns_order = ["City", "Time zone", "Population"]
df = df.reindex(columns=columns_order)
>>> print(df)
```

#### Индексатор `loc[]`

Индексатор `loc[]` позволяет выполнить перестановку столбцов в DataFrame на основе их меток или имен. казав желаемый порядок меток столбцов в индексаторе loc[], мы можем эффективно изменить порядок столбцов в DataFrame:

```python
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012],
"Time zone": ["GMT+3", "GMT+3","GMT+7","GMT+5","GMT+3"]
}

df = pd.DataFrame(data)
columns_order = ["City", "Time zone", "Population"]
df = df.loc[:, columns_order]
print(df)
```

#### Индексатор `iloc[]`

Индексатор `iloc[]` позволяет выполнить перестановку столбцов в DataFrame на основе их целочисленных позиций. По сути он аналогичен аналогично индексатору `loc[]`, но вместо индексации по меткам применяет целочисленные позиции:

```python
data = {
"City": ["Москва", "Санкт-Петербург", "Новосибирск", "Екатеринбург", "Волгоград"],
"Population": [13274, 5653, 1637, 1548, 1012],
"Time zone": ["GMT+3", "GMT+3","GMT+7","GMT+5","GMT+3"]
}

df = pd.DataFrame(data)
columns_order = [0, 2, 1]
df = df.iloc[:, columns_order]
print(df)
```

# Арифметические операции над DataFrame

Арифметические операции над DataFrame представляют распространенную задачу при работе с данными. Будь то вычисление простых сумм, вычисление средних значений или применение более сложных математических операций, Pandas предоставляет ряд методов для эффективного решения этих задач. В целом арифметические операции над DataFrame аналогичны арифметическим операциям над объектами Series. Поскольку каждый столбец в DataFrame по сути является Series, выполнение арифметических операций над DataFrame осуществляется по столбцам.

В Pandas операция сложения (+) используется для поэлементного сложения двух объектов DataFrame или для сложения DataFrame и скалярного значения. Эта операция позволяет изменять данные, в результате чего получается новый DataFrame с вычисленными значениями.

При добавлении скалярного значения к набору данных DataFrame это значение добавляется к каждому элементу в DataFrame. Например, если вы добавите 10 к DataFrame, каждый элемент в DataFrame увеличится на 10:

```python
data = {
"A": [11, 12, 13, 14, 15],
"B": [21, 22, 23, 24, 25]
}

df = pd.DataFrame(data)
# Добавление 10 к каждому элементу в DataFrame
df_10 = df + 10
>>> print(df_10)
    A   B
0  21  31
1  22  32
2  23  33
3  24  34
4  25  35
```

При сложении двух наборов данных DataFrame соответствующие элементы из каждого DataFrame суммируются. Это поэлементное сложение, и для его выполнения оба DataFrame должны иметь одинаковую форму (одинаковое количество строк и столбцов) и столбцы с одинаковыми метками:

```python
data = {
"A": [11, 12, 13, 14, 15],
"B": [21, 22, 23, 24, 25]
}

df = pd.DataFrame(data)
# Поэлементное сложение двух DataFrame
df_twice = df + df

>>> print(df_twice)
    A   B
0  22  42
1  24  44
2  26  46
3  28  48
4  30  50
```

	Операции вычитания, умножения, деления работают аналогично.
