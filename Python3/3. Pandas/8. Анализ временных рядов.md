# Временные метки Timestamp

Для представлении моментов времени в Pandas применяются временные метки. Временная метка - это конкретный момент времени, который обычно содержит дату и временя. Например, "2026-01-19 16:25:00" - это временная метка, которая представляет 16:25 19 январря 2026 года. Для работы с временными метками библиотека Pandas предоставляет объект Timestamp, который является частью модуля datetime в Pandas.

### Создание временных меток

Для создания объектов Timestamp в Pandas можно применять несколько способов: из строки и с помощью набора параметров, которые представляют различные компоненты временной метки. Рассмотрим эти способы.

#### Создание временной метки из строки

Библиотека Pandas может распознавать строку как временную метку. При этом строка должна соответствовать одному из распространенных форматов представления даты и времени:

- YYYY-MM-DD HH:MM:SS
    
- YYYY/MM/DD HH:MM:SS
    
- YYYY.MM.DD HH:MM:SS
    
- YYYY-MM-DD
    
- YYYY/MM/DD
    
- YYYY.MM.DD
    

Для преобразования строки во временную метку Pandas предоставляет функцию pandas.Timestamp(), в которую передается преобразуемая строка и которая возвращает объект `Timestamp`. Пример применения:

```python
import pandas as pd
datatime_str = "2026-01-22 16:30:00"
timestamp = pd.Timestamp(datatime_str)
>>> print(timestamp)
2026-01-22 16:30:00
```

Здесь в качестве формата используется `YYYY-MM-DD HH:MM:SS` - "2026-01-22 16:30:00". Но аналогично мы могли бы использовать другой формат. Например:

```python
datetime_str = "2026/01/22 16:30:00"
timestamp = pd.Timestamp(datetime_str)
>>> print(timestamp)
2026-01-22 16:30:00
```

Pandas также распознает вариации в порядке компонентов даты, включая то, начинается ли дата с года, месяца или дня:

```python
datetime_str = "22-01-2026 16:30:00"
timestamp = pd.Timestamp(datetime_str)
>>> print(timestamp)
2026-02-01 16:30:00
```

#### Использование параметров

Создание объекта `Timestamp` с использованием параметров позволяет указывать отдельные компоненты метки времени, такие как год, месяц, день, час, минута и секунда. Для создания объекта Timestamp с использованием параметров в конструктор `pandas.Timestamp()` передаются значения для каждого из компонентов через следующие параметры:

- `year`: год
    
- `month`: месяц (1-12)
    
- `day`: день (1-31)
    
- `hour`: час (0-23)
    
- `minute`: минуты (0-59)
    
- `second`: секунда (0-59)
    

Этот метод обеспечивает точный контроль над созданием меток времени и особенно полезен, когда имеются отдельные переменные, представляющие компоненты даты и времени:

```python
timestamp = pd.Timestamp(year=2026, month=1, day=22, hour=16, minute=30, second=0)
>>> print(timestamp)    
2026-01-22 16:30:00
```

## Атрибуты и свойства объекта Timestamp

Объект Timestamp предоставляет различные атрибуты и свойства для доступа к различным компонентам метки времени, таким как год, месяц, день, час, минута, секунда и другие. Некоторые распространенные атрибуты и свойства:

- `year`: год
    
- `month`: месяц
    
- `day`: день
    
- `hour`: час
    
- `minute`: минута
    
- `second`: секунда
    
- `microsecond`: микросекунда
    
- `dayofweek`: день недели в виде целого числа, где понедельник равен 0, а воскресенье — 6
    
- `dayofyear`: день года
    
- `is_leap_year`: возвращает логическое значение, указывающее, является ли год временной метки високосным или нет
    

Получение этих атрибутов и свойств:

```python
timestamp = pd.Timestamp(year=2026, month=1, day=22, hour=16, minute=30, second=0)
 
daysofweeks = ["понедельник", "вторник", "среда", "четверг", "пятница", "суббота", "воскресенье"]

print("год високосный: ", "да" if timestamp.is_leap_year else "нет")

```

# Индексирование по дате и времени

Данные временных рядов представляет последовательность значений, записанных через определенные временные интервалы или временные метки. В качестве примеров временных рядов можно назвать цены акций, измерения погоды, показания датчиков, экономические показатели и т.д. Данные временных рядов в Pandas характеризуются следующими аспектами:

- Временная метка: это основная единица измерения в данных временных рядов, которая представляет конкретный момент времени
    
- Частота: временной интервал между последовательными точками данных во временном ряду. Распространенные частоты включают ежедневные, почасовые и ежемесячные.
    
- Индекс даты и времени: в Pandas индекс даты и времени - это специализированный индекс, который позволяет индексировать и сегментировать данные временных рядов по времени. Он состоит из объектов типа Timestamp.
    

Индексирование по дате и времени в Pandas заключается в установке в качестве индекса значений, которые представляет дату и время. Для примера сначала рассмотрим, как создать временной ряд, который затем будет использоваться в качестве индекса:

```python
start_date = '2026-01-01'
end_date = '2026-01-10'
date_index = pd.date_range(start=start_date, end = end_date, freq='D')
>>> print(date_index)
DatetimeIndex(["2026-01-01", "2026-01-02", "2026-01-03", "2026-01-04",
               "2026-01-05", "2026-01-06", "2026-01-07", "2026-01-08",
               "2026-01-09", "2026-01-10"],
              dtype="datetime64[ns]", freq="D")
```

Для создания временного ряда в библиотеке Pandas применяется метод `date_range()`, который генерирует индекс типа DatetimeIndex с фиксированной частотой (последовательность дат и времени). Ее ключевые параметры:

- `start`: левая граница (начальная дата) для генерации дат.
    
- `end`: правая граница (конечная дата) для генерации дат.
    
- `periods`: количество периодов (дат/временных точек) для генерации.
    
- `freq`: частота генерируемых дат, например, "D" (ежедневно), "H" (ежечасно), "M" (конец месяца), "B" (рабочий день) и т. д. По умолчанию - "D".
    

Здесь создается временной ряд, в котором начальная временная метка представляет дату "2026-01-01" (1 января), а конечный временная метка - дату "2026-01-10" (10 января). И всего генерируется 10 подобных точек, которые соответствуют дням - с 1 по 10 января 2026 года (так как для параметра `freq` значение по умолчанию - "D"), что в принципе можно увидеть по консольному выводу.

В примере выше мы автоматически генерировали индекс на основе временных меток, однако, если нам нужно несколько меток не подряд, то мы можем их определить явным образом с помощью функции pandas.to_datetime(), которая принимает список определений временных меток и возвращает индекс:

```python
date_index = pd.to_datetime(["2023-11-16", "2024-01-03", "2024-01-14", "2024-03-08", "2024-05-03", "2026-01-03"])
>>> print(date_index)
DatetimeIndex(["2023-11-16", "2024-01-03", "2024-01-14", "2024-03-08",
               "2024-05-03", "2026-01-03"],
              dtype="datetime64[ns]", freq=None)
```

Затем подобный индекс можно установить для объектов Series или DataFrame.

### Series с индексом в виде даты и времени

Объекты Series в качестве индекса могут использовать временной индекс - объект DatetimeIndex, создание которого было рассмотрено выше. Например, у нас есть набор данных, который содержит ежедневные записи температуры за некоторый период. И мы можем создать серию с индексом даты и времени для представления этих данных:

```python
# Генерируем индекс даты и времени для месяца, начиная с 1 по 10 января 2026 года
start_date = "2026-01-01"
end_date = "2026-01-10"
date_index = pd.date_range(start=start_date, end=end_date, freq="D")
# данные о температуре для каждого дня
data = [-4, -4, -4, -4, -5, -5, -6, -6, -6, -5]
# Создаем Series с индексом даты и времени
temps = pd.Series(data, index=date_index)
>>> print(temps)
2026-01-01   -4
2026-01-02   -4
2026-01-03   -4
2026-01-04   -4
2026-01-05   -5
2026-01-06   -5
2026-01-07   -6
2026-01-08   -6
2026-01-09   -6
2026-01-10   -5
Freq: D, dtype: int64
```

Здесь создается серия с именем temps, которая в качестве индекса использует объект date_index, сгенерированный с помощью функции `pd.date_range()`, а в качестве данных - температуры из массива data. Таким образом, мы получим набор данных, где температура сопоставляется с определенной датой.

### DataFrame с индексом по дате и времени

Аналогичным образом мы можем создавать наборы данных DataFrame с индексом по дате и времени. Например:

Здесь создается DataFrame с индексом по дате и времени. Индекс даты и времени аналогично охватывает период в 10 дней с 1 января 2026 года по 10 января 2026 года с ежедневной частотой. Затем этот индекс связывается с двумя столбцами с примерами данных: "Max Temp" (минимальная температура) и "Min Temp" (максимальная температура). Полученный DataFrame организует и связывает эти данные с временными метками, что упрощает работу с данными временных рядов и их анализ.

# Основные операции с временными рядами

### Индексирование и срезы данных

Библиотека Pandas предоставляет различные способы индексирования и получения срезов данных временных рядов на основе индексов даты и времени:

#### Выбор одной временной метки

Мы можем использовать одно значение даты и времени для выбора данных для определенной временной метки:

```python
start_date = "2026-01-01"
end_date = "2026-01-10"
 
date_index = pd.date_range(start=start_date, end=end_date, freq="D")

# данные о температуре для каждого дня
data = {
    "Max Temp": [-4, -4, -4, -4, -5, -5, -6, -6, -6, -5],
    "Min Temp": [-8, -9, -9, -10, -10, -12, -11, -10, -10, -10]
}

# Создаем DataFrame с индексом даты и времени
df = pd.DataFrame(data, index=date_index)
 
# Выбор данных для определенной даты
january_8 = df.loc["2026-01-08"]
>>> print(january_8)
Max Temp    -6
Min Temp   -10
Name: 2026-01-08 00:00:00, dtype: int64
```

Здесь мы создаем датафрейм df с данными по минимальным и максимальным температурам на отрезке с 1 по 10 января 2026 года. Причем этот датафрейм используем индексацию по дате и времени. Затем мы используем индексатор `loc[]` для выбора данных из набора данных df для определенной даты - "2026-01-08" (то есть 8 января 2026 года). В итоге мы получим показатели минимальной/максимальнйо температуры за данный конкретный день.

#### Выбор диапазона дат

Аналогично мы можем выбрать данные для определенного диапазона дат, используя срезы:

```python
start_date = "2026-01-01"

# Создаем DataFrame с индексом даты и времени
df = pd.DataFrame(data, index=date_index)
 
january_4_8 = df.loc["2026-01-04":"2026-01-08"]
>>> print(january_4_8)
```

Здесь, используя синтаксис срезов на DataFrame, мы выбираем данных для определенного диапазона дат, а именно с "2026-01-04" до "2026-01-08". В результате мы получим DataFrame, в котором отобразятся данные о температурах за период с 4 января по 8 января 2026 года.

#### Выбор по году, месяцу или дню

С помощью индексатора `loc[]` также можно выбрать данных за определенные годы, месяцы или дни. Например:

```python
# Создание индекса даты и времени с различными датами

data = {

january_3 = df.loc[(df.index.day == 3) & (df.index.month == 1)]

print(january_3)
 
# Выбор данных за 2024 год
year_2024 = df.loc["2024"]

print(year_2024)
```

Прежде всего здесь генерируем вручную индекс на основе временных меток:

```python
date_index = pd.to_datetime(["2023-11-16", "2024-01-03", "2024-01-14", "2024-03-08", "2024-05-03", "2026-01-03"])
```

```python
january_2024 = df.loc["2024-01"]
```

```python
january_3 = df.loc[(df.index.day == 3) & (df.index.month == 1)]
```

Поскольку здесь индекс в датафрейме представляет объект Timestamp, то через свойства `index.day` и `index.month` мы можем получить день и год, определить два условия и через логическую операцию & объединить их.

В конце выбираем данные за 2024 год с помощью `loc[]`:

```python
year_2024 = df.loc["2024"]
```

# Изменение частоты данных

Изменение частоты данных или resampling представляет процесс изменения количества данных временного ряда. Можно понижать или повышать частоту в зависимости от потребностей анализа. Для выполнения повтрной выборки библиотека Pandas предоставляет метод resample():

```python
DataFrame.resample(rule, closed=None, label=None, convention="start", on=None, level=None, origin="start_day", offset=None, group_keys=False)
```

Из параметров метода отмечу лишь первый параметр, который является обязательным и который обычно представляет строку и который указывает на тип преобразования.
#### Уменьшение частоты

Понижение частоты данных временного ряда заключается в агрегировании или суммировании значений за более длительные периоды времени. Это может быть полезно для упрощения данных, уменьшения шума и получения более широкого обзора тенденций и закономерностей за более длительные временные интервалы.

Например, у нас есть ежедневные данные о температуре, и мы хотим свести их к недельным средним значениям для более плавного обзора температурных тенденций:

```python
# Генерируем индекс даты и времени для месяца, начиная с 1 по 15 января 2026 года

end_date = "2026-01-15"
```

#### Повышение частоты

Повышение частоты представляет увеличение частоты данных временных точек, которое обычно происходит путем интерполяции или генерации новых точек данных между существующими временными точками. Основная цель повышения частоты дискретизации - обеспечить более детальное или мелкозернистое представление данных с более высоким временным разрешением. Увеличение выборки полезно, когда необходимо повысить детализацию данных временного ряда или выровнять их с данными более высокой частоты для дальнейшего анализа или моделирования.

Например, у нас есть объект DataFrame с ежемесячными данными о температуре за январь, февраль и март 2025 года, и нам надо из них получить ежедвневные температуры:

```python
data = {
```

# Сдвиг данных

В анализе временных рядов "сдвиг" (shifting) или "запаздывание" (lagging) относится к операции перемещения точек данных вперед или назад во времени внутри временного ряда.

Запаздывание (lag) представляет количество временных периодов, на которые данные сдвигаются назад во времени. Положительное значение запаздывания означает сдвиг данных назад, в прошлое, а отрицательное запаздывание (lead) представляет сдвиг данных вперед, в будущее.

Сдвиг и запаздывание позволяют создать новые наборов данных временных рядов или могут применяться для проведения сравнений по времени.

Для определения сдвига в Pandas применяется метод shift()

Пусть у нас есть DataFrame временного ряда с данными о температуре, и мы хотим определить запаздающий признак в 1 временной период:

```python
data = {
    "Date": ["2026-01-01", "2026-01-02", "2026-01-03", "2026-01-04", "2026-01-05", "2026-01-06", "2026-01-07"],
    "Temp": [-4, -4, -4, -4, -5, -5, -6]

df.set_index("Date", inplace=True)

df["Temp_lag1"] = df["Temp"].shift(1)
```

# Скользящие окна

Скользящие окна (Rolling Windows) представляют один из методов анализа временных рядов. Он подразумевает создание подмножеств данных путем "перемещения" окна или временного интервала фиксированного размера по временному ряду. В каждом окне могут применяться различные вычисления или операции, такие как вычисление скользящих средних, стандартных отклонений или других статистических показателей.

Скользящие окна могут помочь сгладить зашумленные или изменчивые данные временных рядов путем вычисления скользящих средних или других агрегатов внутри окна. Они используются для выявления трендов, закономерностей или аномалий в данных временных рядов путем анализа статистических данных, вычисленных внутри скользящего окна.

### Скользящая средняя

Скользящая средная (Rolling Mean / Moving Average) вычисляет среднее значение временного ряда в пределах скользящего окна заданного размера. Это скользящее окно перемещается вдоль временного ряда и в каждой точке времени вычисляет среднее значение точек данных внутри этого окна.

Для вычисления скользящей средней Pandas предоставляет метод DataFrame.rolling(), который создает скользящее окно определенного размера:

```python
DataFrame.rolling(window, min_periods=None, center=False, win_type=None, on=None, closed=None, step=None, method="single")
```

Ключевым у метода является первый, он же единственный обязательный параметр, который устанавливает интервал окна. Рассмотрим применение метода на примере:

```python
data = {
    "Date": ["2026-01-01", "2026-01-02", "2026-01-03", "2026-01-04", "2026-01-05", "2026-01-06", "2026-01-07"],
    "Temp": [-4, -4, -4, -4, -5, -5, -6]
}
df = pd.DataFrame(data)
# Преобразование данных столбца "Date" в Timestamp
df["Date"] = pd.to_datetime(df["Date"])
# Установка столбца "Date" в качестве индекса
df.set_index("Date", inplace=True)
# Вычисление скользящей средней за 3 дня для данных о суточной температуре
rolling_mean = df["Temp"].rolling(window=3).mean()
>>> print(rolling_mean)

2026-01-04   -4.000000
2026-01-05   -4.333333
2026-01-06   -4.666667
2026-01-07   -5.333333
Name: Temp, dtype: float64
```

Здесь мы имеем набор температур на 7 дней с 1 по 7 января 2026 года и вычисляем для этого столбца среднюю скользящюю с помощью выражения `rolling(window=3)`. В данном случае используется скользащее окно на 3 периода (в нашем случае 3 дня). Это означает, что для каждого дня скользящее окно включает текущий день и два предыдущих дня. Если бы у нас временные метки располагались друг от друга на расстоянии месяца, то параметр `window=3` означал бы скользящее окно длиной в три месяца.

Затем мы применяем метод `mean()` для вычисления средней температуры внутри скользящего окна. В итоге полученный объект rolling_mean будет содержать скользящую среднюю за 3 дня для данных о суточной температуре.

Причем по консольному выводу видно, что скользящая средняя начинается со значения NaN в течение первых двух дней, поскольку недостаточно точек данных для ее вычисления. Начиная с третьего дня, происходит сглаженное вычисление данных о температуре, выделяя тенденции и закономерности и уменьшая влияние суточных колебаний.

### Скользящее стандартное отклонение

Скользящее стандартное отклонение (rolling standard deviation) вычисляет стандартное отклонение временного ряда в скользящем окне. Это скользящее окно перемещается вдоль временного ряда, и в каждой точке времени вычисляется стандартное отклонение точек данных внутри этого окна.

Для вычисления отклонения Pandas предоставляет функцию std(), которая применяется к набору средних скользящих:

```python
data = {

df.set_index("Date", inplace=True)
 
# Вычисление 3-дневного скользящего стандартного отклонения для ежедневных данных о температуре
```

```python
df.set_index("Date", inplace=True)

2026-01-07    2250.0
Name: Sales, dtype: float64
```

Здесь набор данных df содержит данные о ежедневных продажах (столбец "Sales"). И в данном случае мы вычисляем скользящую сумму за 3 дня на основе данных о ежедневных продажах. То есть для каждого дня скользящее окно включает текущий день и два предыдущих дня и суммирует продажи за эти дни.

Скользящая сумма полезна для отслеживания накопления значений во времени, таких как общие продажи, уровни запасов или другие кумулятивные показатели. Это может помочь в выявлении тенденций или закономерностей, связанных с кумулятивными данными в анализе временных рядов.

### Экспоненциальная скользящая средняя

Экспоненциальная скользящая средняя (Exponential Moving Average или EMA) - это тип скользящей средней, который присваивает значениям во временном ряду экспоненциально убывающие веса. Она особенно полезна для выявления последних тенденций и более чувствительна к свежим данным по сравнению с простой скользящей средней.

Для вычисления экспоненциальной скользящей средней в Pandas применяется метод ewm():

```python
DataFrame.ewm(com=None, span=None, halflife=None, alpha=None, min_periods=0, adjust=True, ignore_na=False, times=None, method="single")
```

Из всех параметров здесь следует выделить параметр `span`, который устанавливает интервал для экспоненциальной скользящей средней. Интервал представляет собой коэффициент сглаживания или количество временных периодов, за которые вычисляется EMA. Рассмотрим применение этой функции:

```python
data = {
    "Date": ["2026-01-01", "2026-01-02", "2026-01-03", "2026-01-04", "2026-01-05", "2026-01-06", "2026-01-07"],
    "Price": [500, 700, 600, 900, 850, 800, 600]
}
df = pd.DataFrame(data)
# Преобразование данных столбца "Date" в Timestamp
df["Date"] = pd.to_datetime(df["Date"])
# Установка столбца "Date" в качестве индекса
df.set_index("Date", inplace=True)
# Вычисление 3-дневной экспоненциальной скользящей средней цен за день
prices_ema = df["Price"].ewm(span=3).mean()
>>> print(prices_ema)

Name: Price, dtype: float64
```

Здесь датафрейм df содержит данные о некоторых ценах. Для вычисления экспоненциальной скользящей средней для столбца "Price" выполняем цепочку методов `df["Price"].ewm(span=3).mean()`. Здесь вызов `ewm(span=3)` создает скользящее окно с интервалом в три дня. Затем применяем метод `` mean()` `` для вычисления экспоненциальной скользящей средней цен за день в пределах указанного окна.
