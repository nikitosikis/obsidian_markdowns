
# Создание и свойства Series

Series представляет одномерный массив с метками, который может хранить данные любого типа (целые числа, строки, числа с плавающей запятой, объекты Python и многое другое). Он состоит из двух основных компонентов: собственно данных и индекса.

Данные в Series представляет значения или элементы любого типа данных, например, числа, текст или даже более сложные объекты. Данные могут быть предоставлены в виде:

- словаря Python (dict)
    
- списка Python (list)
    
- кортежа Python
    
- массива NumPy (ndarray)
    
- скалярных значений
    
- других структур данных
    

При чем объект Series имеет фиксированный размер и форму после создания. Он может содержать лишь определенное количество элементов, которое является неизменяемым. При наличии пропущенных данных объект Series представляет их как значения NaN (Not-a-Number), предоставляя методы для эффективной обработки и манипулирования пропущенными значениями. При этом надо понимать, что серии, хотя и похожи на списки Python, которые могут содержать элементы разных типов данных, объекты Series больше подходят для работы с однородными данными.

Индекс в Series представляет последовательность меток, которые идентифицируют каждый элемент в Series. По умолчанию индекс начинается с 0 и увеличивается на 1, но мы можем настроить его и использовать в качестве индекса данные любого хешируемого типа, в том числе строк или дат. Индекс обеспечивает интуитивно понятный доступ к данным в серии. При этом объект Series автоматически выравнивает свои данные на основе меток индексов, обеспечивая эффективные и согласованные операции даже при работе с несколькими объектами Series.

# Создание объекта Series

Для создания объекта применяется конструктур pandas.Series()

```python
import pandas as pd
pd.Series(data=None, index=None, dtype=None, name=None, copy=None)
```

Здесь ключевые два параметра. Параметр `data` - данные, на основании которых строится объект, и параметр `index`, который указывает, как индексировать данные. Рассмотрим различные способы создания серии.

## Создание Series на основе списка Python

```python
series1 = pd.Series([1,2,3,4,5])
>>> print(series1)
0    1
1    2
2    3
3    4
4    5
dtype: int64
```

Тут стоит отметить, что в левой колонке вывода отображается индекс объекта Series. По умолчанию при создании объекта Series из списка индекс генерируется автоматически как последовательность целых чисел, начиная с 0. А в правой колонке вывода отображаются собственно значения объекта Series. Эти значения были созданы из элементов списка.

Внизу отображается тип данных (свойство dtype)объекта Series. Этот объект Series содержит целые числа, поэтому тип данных - `int64` (для целых чисел pandas по умолчанию использует тип `int64`).

## Создание Series из массива NumPy

```python
import numpy as np
my_array = np.array([1,2,3,4,5], int)
series = pd.Series(my_array)
>>> print(series)
0    1
1    2
2    3
3    4
4    5
dtype: int64
```

## Создание Series из словаря Python

Словарь представляет неупорядоченную коллекцию элементов, где каждый элемент хранится в виде пары ключ-значение. Ключи в словаре уникальны и используются для доступа к соответствующим значениям. При создании серии ключи словаря становятся индексами (или метками) серии, а значения словаря - соответствующими значениями в серии. Например:

```python
series = pd.Series({'b':1,'a':0,'c':'2'})
>>>print(series)
b    1
a    0
c    2
dtype: int64
```

## Создание Series из кортежа Python

Кортеж представляет набор объектов Python, разделенных запятыми. Кортеж похож на список. Разница между ними заключается в том, что кортеж неизменяем, то есть мы не можем изменить элементы кортежа после того, как он был присвоен. Создание серии из кортежа:

```python
series1 = pd.Series((1,2,3,4,5))
>>> print(series1)
0    1
1    2
2    3
3    4
4    5
dtype: int64
```

## Содание серии на основе скалярного значения

Мы можем создать серию с одним или несколькими элементами, которые будут представлять определенное значение:

```python
series = pd.Series(15)
>>> print(series)
0    15
dtype: int64
```

## Применение индексов

В примере выше мы использовали только один (первый) параметр - сами данные. Но параметр index позволяет более точно настроить сопоставление данных с индексами. Этот параметр представляет метки или имена, присвоенные каждому элементу в серии. Он помогает идентифицировать и получать доступ к конкретным элементам Series. По умолчанию индекс начинается с 0 и увеличивается на 1 для каждого элемента, но мы можем настроить его на любую последовательность меток. Например, мы хотим создать серию с одним и тем же значением, которое повторяется несколько раз (допустим, значение 15, которое повторяется 5 раз):

```python
value = 15
series = pd.Series(value, index=[0,1,2,3,4])
>>> print(series)
0    15
1    15
2    15
3    15
4    15
dtype: int64
```

Аналогичным образом мы можем настроить произвольные индексы на произвольные элементы. Причем в качестве индексов необязательно должны выступать числа. Например, пусть у нас есть список возрастов разных людей, и мы хотим присвоить каждому возрасту определенную метку. В качестве подобной метки можно взять имя соответствующего человека:

```python
ages = [41,56,31]
names=['Tom','Bob','Sam']
series = pd.Series(ages, index=names)
>>> print(series)
Tom    41
Bob    56
Sam    31
dtype: int64
```

# Обращение к элементам серии по индексу

Индексы играют важную роль, прежде всего потому, что позволяют обращаться к элементам в серии подобно тому, как мы можем обращаться к элементам списка в Python:

```python
series = pd.Series([11,12,13,14,15])
>>> print(series[1])
12
>>> series[1] = 121
print(series)
0     11
1    121
2     13
3     14
4     15
dtype: int64
```

В примере выше индексы устанавливались автоматически - начиная с 0. Явная установка индексов позволяет получить больший контроль и придать элементам серии более читабельные метки:

```python
ages = [41,56,31]
names=['Tom','Bob','Sam']
series = pd.Series(ages, index=names)
>>> print(series['Tom'])
41
```

Причем, используя набор индексов, мы можем получить часть серии, которая состоит из соответствующих элементов:

```python
series = pd.Series([11,12,13,14,15])
sub_series = series[[0,2,3]]
>>> print(sub_series)
0    11
2    13
3    14
dtype: int64
```

## Свойства серии

Объект Series обладает рядом свойств, которые могут потребоваться при работе с сериями Pandas:

- Имя серии
    
- Размер серии
    
- Уникальность элементов
    
- Количество значений в серии
    
- Тип данных
    

Рассмотрим эти свойства.

### Имя серии

Присвоение имени объекту Series повышает ясность и контекст данных, улучшает читаемость кода, облегчает интерпретацию данных и упрощает совместную работу. В pandas существует несколько способов присвоить имя объекту Series. Несколько распространенных подходов:

- Присвоение имени при инициализации

```python
series = pd.Series([11,12,13,14,15], name='MySeries')
```

- Присвоение имени с помощью атрибута name

```python
series = pd.Series([11,12,13,14,15])
series.name = 'MySeries'
```

### Размер серии

азмер серии представляет общее количество элементов, которые она содержит. Размер серии можно получить с помощью функции len() или атрибута size:

```python
series = pd.Series([11,12,13,14,15])
>>> print(len(series))
5
>>> print(series.size)
5
```

### Уникальность элементов серии

Уникальность позволяет идентифицировать уникальные значения в серии и понимать разнообразие данных. Мы можем проверить уникальность серии, используя метод unique(), который возвращает массив уникальных значений:

```python
series = pd.Series([10,10,10,12,12])
unique_values = series.unique()
>>> print(unique_values)
[10, 12]
```

### Количество значений в серии

Количество значений представляет собой частоту каждого уникального значения в серии. Это помогает понять распределение и появление различных значений. Для вычисления частоты применяется метод value_counts(), который создает новую серию, где каждому уникальному значению в исходной серии соответствует его количество:

```python
series = pd.Series([10,20,30,40,50,10,20,30,20])

value_counts = series.value_counts()
>>> print(value_counts)
20    3
10    2
30    2
40    1
50    1
Name: count, dtype: int64
```

### Типы данных

Структура данных `Series` может хранить данные различных типов, таких как числа, строки, логические значения и другие. Типы данных указывают, как данные хранятся и какие операции над ними могут выполняться. Pandas заимствует свои типы данных из другой библиотеки Python - NumPy и также имеет некоторые свои специфичные типы данных. При определении объектов `Series` Pandas автоматически присваивает подходящий тип данных каждому столбцу или серии на основе содержащихся в них значений. Но, используя аргумент `dtype` в конструкторе `Series`, мы можем явным образом указать тип данных. Рассмотрим некоторые основные типы данных.

- Object
    
    Тип данных `Object` используется для хранения любых данных, кроме числовых и логических. Он может хранить строки, комплексные типы или объекты Python. Тип данных `Object` также используется, когда pandas не может определить конкретный тип данных для столбца или серии.

- int64

Тип данных int64 используется для хранения 64-битных целых чисел: значения от -9223372036854775808 до 9223372036854775807. Это тип данных применяется по умолчанию для числовых столбцов или серий, которые не содержат десятичных знаков или пропущенных значений.

- Float64

Тип данных **float64** представляет 64-разрядные числа с плавающей запятой и может хранить десятичные числа с точностью до 15 знаков после запятой. Это тип данных применяется по умолчанию для числовых столбцов или серий, которые содержат десятичные знаки или пропущенные значения. Если нам надо указать отсутствие значения (или точнее Not a Number), мы можем использовать константу np.nan из библиотеки numpy.

- Логические значения bool

Логический тип данных (bool) используется для хранения значений `True` или `False`. Это тип данных по умолчанию применяется для столбцов или серий, которые содержат только значения `True` или `False`.

# Выборка данных из Series

При работе с данными крайне важно быстро получить общее представление о наборе данных, прежде чем приступать к дальнейшему анализу. Pandas предоставляет несколько методов для просмотра данных, включая `head()`, `tail()` и `sample()`. Эти методы позволяют изучать начальные строки, последние строки и получать случайные выборки из Series.

### Метод iloc

Метод iloc() позволяет получить элементы или подмножества на основе их позиций или условии, которое зависит от позиции. Например:

```python
ages = [41,56,31,38]
names = ['Tom','Bob','Sam','Alice']
series = pd.Series(ages, index=names)

# доступ к одному элементу по позиции
element = series.iloc[2]

subset = series.iloc[[0, 2]]
>>> print subset
Tom    41
Sam    31
dtype: int64
```

Метод `iloc()` позволяет нам получать доступ к конкретным элементам на основе их позиций, даже если в серии используются нечисловые индексы, как в случае выше. Использование `iloc[2]` извлекает элемент на позиции 2, который в примере выше равен 31. Аналогично, `series.iloc[[0, 2]]` возвращает подмножество Series, которое содержит элементы на позициях 0 и 2.

### Метод head()

Метод head() позволяет просмотреть первые несколько строк данных. По умолчанию отображаются первые 5 строк, но с помощью необязательного параметра можно указать конкретное количество отображаемых строк:

```python
series = pd.Series([10,20,30,40,50,10,20,30,20])
print(series.head(3))
0    10
1    20
2    30
dtype: int64
```

### Метод tail()

Метод tail() позволяет просмотреть последние несколько строк. По умолчанию отображаются последние 5 строк, но также с помощью необязательного параметра можно указать конкретное количество отображаемых строк:

```python
series = pd.Series([10,20,30,40,50,10,20,30,20])
print(series.tail(3))
6    20
7    30
8    20
dtype: int64
```

### Метод Sample

Метод sample() позволяет получать случайные выборки из данных. В качестве параметра метод принимает количество извлекаемых выборок, что позволяет проводить рандомизированное исследование набора данных:

```python

series = pd.Series([10,20,30,40,50,10,20,30,20])
sample_data = series.sample(3)
1    20
3    40
4    50
dtype: int64

```

# Арифметические операции с Series

Объект Series предоставляет широкий спектр функций и методов для обработки и анализа данных. Прежде всего рассмотрим арифметические операции. Подобно арифметическим операциям над числами, Pandas позволяет выполнять эти операции над Series, позволяя складывать, вычитать, умножать или делить целые массивы данных за одну операцию. Также имеется поддержка более сложных математических операций и функций.

Оператор сложения позволяет складывать поэлементно два объекта Series одинаковой длины. 

```python
s1 = pd.Series([1,2,3,4,5])
s2 = pd.Series([6,7,8,9,10])

s3 = s1 + s2
>>> print(s3)
0     7
1     9
2    11
3    13
4    15
dtype: int64
```

Вычитание, умножение и деление, остаток от деления и деление с остатком, как и сложение, и работают поэлементно.

Кроме того, можно выполнять скалярное сложение(умножение, деление и т.д.), когда к каждому элементу объекта Series добавляется скалярное значение. Например:

```python
s1 = pd.Series([1,2,3,4,5])
s2 = s1 + 10
>>> print(s2)
0    11
1    12
2    13
3    14
4    15
dtype: int64
```

В случае, если операция сложения применяется к строкам, то происходит конкатенация строк. Например:

```python
s1 = pd.Series(['a','b','c'])
s2 = pd.Series(['d','e','f'])

s3 = s1 + s2
>>> print(s3)
0    ad
1    be
2    cf
dtype: object
```

### Операции над сериями разной длины

При попытке выполнить арифметические операции над двумя рядами разной длины Pandas заполняет недостающие элементы значениями NaN. Это может создать потенциальные проблемы, поскольку `NaN` - фактически это число с плавающей точкой, и, следовательно, весь ряд будет преобразован в число с плавающей запятой, даже если изначально это был целочисленный ряд. Например:

```python
s1 = pd.Series([1,2,3,4,5])
s2 = pd.Series([6,7,8])
0     7.0
1     9.0
2    11.0
3    NaN
4    NaN
dtype: float64
```

Здесь индексы 3 и 4 имеют значение NaN, поскольку операция не может быть выполнена из-за отсутствия соответствующих элементов в серии s2.

# Фильтрация

Одной из ключевых задач при работе с данными является фильтрация на основе определенных условий или критериев. Pandas предоставляет богатые возможности для фильтрации данных в Series.

### Фильтрация с помощью операторов сравнения

Прежде всего для фильтрации данных мы можем использовать такие операторы сравнения, как `>, >, <=, >=, ==` и `!=`. Эти операторы позволяют создать логические маски, которые указывают, удовлетворяет ли каждый элемент в Series указанному условию. Рассмотрим несколько примеров:

```python
s = pd.Series([10,20,30,40,50])

filtered_data = s[s > 30]
>>> print(filtered_data)
3    40
4    50
dtype: int64
```

Здесь, используя оператор сравнения `<`, мы создаем логическую маску, которая проверяет, превышает ли каждый элемент значение 30. Применение этой маски к Series (`s > 30`) возвращает отфильтрованные данные, которые удовлетворяют условию.

Другой пример:

```python
s = pd.Series(['Tom','Bob','Sam','Tom'])
filtered_data = s[s == 'Tom']
>>> print(filtered_data)
0    Tom
3    Tom
dtype: object
```

Здесь с помощью оператора сравнения `==` создаем логическую маску, которая проверяет, равен ли каждый элемент значению "Tom". Применение этой маски (`s == "Tom"`) фильтрует Series, сохраняя только те элементы, которые соответствуют указанному значению.

Аналогично можно использовать другие операторы сравнения.

### Фильтрация с помощью логических операторов

Логические операторы, (`&` и `|`) позволяют объединить нескольких условий. Как и в целом в Python, оператор & указывает, что элементы должны соответствовать сразу двум условиям. Например, нам надо отфильтровать элементы, которые одновременно больше 8 и меньше 20:

```python
s = pd.Series([1,2,3,4,5,8,13,21,35,55,89])

filtered_data = s[(s > 8) & (s < 30)]
>>> print(filtered_data)
6 13
7 21
dtype: int64
```

Здесь объединяем два условия: `(s > 8) & (s < 30)`. Эти условия создают две отдельные логические маски. Первая маска проверяет, больше ли каждый элемент 8, а вторая маска проверяет, меньше ли каждый элемент 30. Затем мы объединяем эти две маски с помощью логического оператора & внутри квадратных скобок: `[(s > 8) & (s < 30)]`. Логический оператор & оценивает условия для каждого элемента и возвращает логическую маску, которая указывает, какие элементы удовлетворяют одновременно обоим условиям.

Другой пример: если нам надо определить, соответствуют значения хотя бы одному из двух условий, то мы можем использовать оператор `|`:

```python
s = pd.Series([1,2,3,4,5,8,13,21,35,55,89])

filtered_data = s[(s > 8) | (s < 30)]
>>> print(filtered_data)
0     1
1     2
2     3
3     5
7    34
8    55
9    89
dtype: int64
```

Здесь для фильтрации объединяем две маски с помощью логического оператора | в квадратных скобках: `[(s < 8) | (s > 30)]`. Логический оператор | оценивает условия для каждого элемента и возвращает логическую маску, которая указывает, какие элементы удовлетворяют как минимум одному из условий.

### Методы фильтрации

Также Pandas предоставляет ряд встроенных методов для фильтрации данных. Рассмотрим некоторые из них.

Метод isin() позволяет проверить, соответствуют ли элементы в Series каким-либо из указанных значений. Этот метод полезен, когда необходимо отфильтровать данные на основе нескольких значений. Например:

```python
s = pd.Series(["Tom", "Bob", "Sam", "Kate", "Alex", "Alice"])
filtered_data = s[s.isin(["Tom", "Rob", "Alice"])] 

>>> print(filtered_data)
0      Tom
5    Alice
dtype: object
```

Здесь метод `isin()` проверяет, присутствует ли каждый элемент в Series в списке значений `["Tom", "Rob", "Alice"]`. В итоге мы получаем только те элементы, которые есть в этом списке.

Еще один полезный метод - метод between() помогает отфильтровать данные, которые располагаются в определенном диапазоне. Например, у нас есть числовой ряд, и нам надо получить числа из диапазона от 8 до 30:

```python
s = pd.Series([1,2,3,4,5,8,13,21,35,55,89])

filtered_data = s[s.between(8,30)]
>>> print(filtered_data)
5 8
6 13
7 21
dtype: int64
```
