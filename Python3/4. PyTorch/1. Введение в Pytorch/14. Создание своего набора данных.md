
В предыдущих статьях мы рассмотрели, как использовать встроенный набор данных (датасет) для обучения нейронной сети. Теперь же рассмотрим, как определить свой собственный набор данных. Если мы хотим создать свой собственный набор данных, то нам надо определить класс-наследник типа torch.utils.data.Dataset. И в этом классе необходимо реализовывать три функции:

- __init__: стандартная функция инициализатора, которая выполняется один раз при создании объекта `Dataset`.
    
- __len__: возвращает количество образцов в наборе данных.
    
- __getitem__: загружает и возвращает образец из набора данных по заданному индексу.
    

К примеру, создадим свой набор данных на основе изображений. Пусть сами файлы изображений располагаются в проекте в папке images, а их метки - в файле labels.csv

Допустим, в папке images имеются следующие файлы изображений:

```
coat1.jpg
coat2.jpg
coat3.jpg
shirt1.jpg
shirt2.jpg
shirt3.jpg
switer1.jpg
switer2.jpg
switer3.jpg
trouser1.jpg
trouser2.jpg
trouser3.jpg
tshirt1.jpg
tshirt2.jpg
tshirt3.jpg
```

По поводу типа изображений стоит отметить, что теоретически изображения могут отличаться по размеру, но тем не менее обычно нейронные сети требуют одинаковых размеров для всех изображений в батче. Поэтому предполагается, что изображения имеют один и тот же размер и имеют одно и то же количество каналогв (1 - для серых, 3/4 - для цветных RGB). В принципе можно и непосредственно при загрузке изображений программным образом их обрабатывать и приводить к нужной форме. Но в данном случае предположим, что изображения уже имеют одинаковые размеры.

Тогда файл labels.csv выглядит следующим образом:

```csv
coat1.jpg, 0
coat2.jpg, 0
coat3.jpg, 0
shirt1.jpg, 1
shirt2.jpg, 1
shirt3.jpg, 1
switer1.jpg, 2
switer2.jpg, 2
switer3.jpg, 2
trouser1.jpg, 3
trouser2.jpg, 3
trouser3.jpg, 3
tshirt1.jpg, 4
tshirt2.jpg, 4
tshirt3.jpg, 4
```

То есть на каждой строке указана информация об отдельном изображении в формате `[имя_файла], [номер_категории/метки]`. То есть в примере выше 5 категорий: от 0 до 4.

Для работы с данными определим следующий класс:

```python
import os
import torch
from torch.utils.data import Dataset
from torchvision.io import decode_image
import pandas as pd
 
class MyImageDataset(Dataset):
    def __init__(self, labels_file, img_dir, transform=None, target_transform=None):
        self.img_labels = pd.read_csv(labels_file)
        self.img_dir = img_dir
        self.transform = transform
        self.target_transform = target_transform
 
    def __len__(self):
        return len(self.img_labels)
 
    def __getitem__(self, idx):
        img_path = os.path.join(self.img_dir, self.img_labels.iloc[idx, 0])
        image = decode_image(img_path)      # преобразуем изображение в тензор
        label = self.img_labels.iloc[idx, 1]
        if self.transform:   # применяем преобразования к изображению, если они заданы
            image = self.transform(image)
        if self.target_transform:   # применяем преобразования к метке, если они заданы
            label = self.target_transform(label)
        return image, label
```

Рассмотрим основные компоненты. Вначале импортируем все необходимые библиотеки:

- `os`: для работы с файловой системой
    
- `Dataset`: базовый класс для создания датасетов
    
- `decode_image`: для декодирования изображений
    
- `pandas`: для работы с CSV файлами
    

Класс `MyImageDataset` наследуется от torch.utils.data.Dataset и реализует 3 ранее упомянутых метода класса:

- Метод __init__. Здесь мы инициализируем каталог, который содержит изображения, файл аннотаций и оба типа преобразований:

```python
def __init__(self, labels_file, img_dir, transform=None, target_transform=None):
    self.img_labels = pd.read_csv(labels_file)
    self.img_dir = img_dir
    self.transform = transform
    self.target_transform = target_transform
```

- Параметры метода:
    
    - `labels_file`: путь к CSV файлу с метками
        
    - `img_dir`: путь к папке с изображениями
        
    - `transform`: преобразования для изображений (например, `torchvision.transforms.ToTensor()`)
        
    - `target_transform`: преобразования для меток
        
    
    В итоге этот метод загружает CSV файл с метками в DataFrame и сохраняет пути и параметры преобразований
    
- Метод __len__ просто возвращает количество изображений в датасете (равное количеству строк в CSV):

```python
def __len__(self):
    return len(self.img_labels)
```

- Метод __getitem__() - ключевой метод, который загружает и возвращает изображение из набора данных по заданному индексу idx. На основе индекса она определяет местоположение изображения на диске, преобразует его в тензор с помощью функции `decode_image`, извлекает соответствующую метку из данных CSV в `self.img_labels`, вызывает функции преобразования (если они применяются) и возвращает тензорное изображение и соответствующую ему метку в виде кортежа:

```python
def __getitem__(self, idx):
    img_path = os.path.join(self.img_dir, self.img_labels.iloc[idx, 0])
    image = decode_image(img_path)      # преобразуем изображение в тензор
    label = self.img_labels.iloc[idx, 1]
    if self.transform:   # применяем преобразования к изображению, если они заданы
        image = self.transform(image)
    if self.target_transform:   # применяем преобразования к метке, если они заданы
        label = self.target_transform(label)
    return image, label
```

Здесь надо отметить, что для преобразования изображения в тензор применяется встроенная функция decode_image из torchvision, которая выполняет декодирование изображения в тензор типа `uint8` из пути к изображению или из необработанных закодированных байтов. В настоящее время она поддерживает такие форматы изображений, как `jpeg, png, gif` и `webp`. Значения выходного тензора в большинстве случаев имеют тип `uint8` и находятся в диапазоне `[0, 255]`.

Если изображение представляет собой 16-битный `png`, то выходной тензор имеет тип `uint16` и находится в диапазоне `[0, 65535]`. Поскольку поддержка `uint16` в pytorch ограничена, то рекомендуется после этой функции вызвать `torchvision.transforms.v2.functional.to_dtype()` с параметром `scale=True`, чтобы преобразовать декодированное изображение в тензор типа `uint8` или `float`.

И также стоит отметить, что CSV файл должен содержать в первом столбце имена файлов изображений, а во втором - метки (как было показано выше).

```python
import torch

from torchvision.io import decode_image
import matplotlib.pyplot as plt

class MyImageDataset(Dataset):
```

Разберем пошагово основные аспекты кода:

- Определение меток и подготовка графика

```python
labels_map = {

}
figure = plt.figure(figsize=(8, 8))

```

- Здесь выполняются следующие действия:
    
    1. Создается словарь `labels_map`, который сопоставляет числовые метки (от 0 до 9) из набора данных с соответствующими текстовыми названиями категорий (например, 0: "T-Shirt", 9: "Ankle Boot").
        
    2. Далее создается новая фигура Matplotlib (`figure`) с размером 8x8, которая будет служить контейнером для отображения изображений.
        
    3. Определяются переменные `cols` и `rows` (3 и 3), устанавливая размер сетки для отображения изображений (3x3 = 9 изображений).
        
- На следующем шаге выполняется визуализация случайных образцов

```python
for i in range(1, cols * rows + 1):

    img, label = data[sample_idx]
    figure.add_subplot(rows, cols, i)
```

Цикл выполняет следующие действия 9 раз (для 3x3 ячеек):

1. `sample_idx = torch.randint(...)`: случайным образом выбирается индекс (`sample_idx`) из обучающего набора данных.
    
2. `img, label = training_data[sample_idx]`: по выбранному индексу из набора данных извлекаются изображение (`img`) и его метка (`label`).
    
3. `figure.add_subplot(rows, cols, i)`: добавляется новая область к фигуре в текущую позицию сетки.
    
4. `plt.title(labels_map[label])`: устанавливается заголовок с помощью текстовой метки из `labels_map`.
    
5. `plt.axis("off")`: скрываются оси x и y, чтобы отображалось только изображение.
    
6. `img = img.permute(1, 2, 0)`. При выполнении программы мы можем столкнуться с ошибкой типа:

```python

```

"Invalid shape" указывает, что изображение-тензор имеет некорректную форму для отображения с помощью matplotlib.
    
Изображение загружается через `decode_image()` с формой `[C, H, W]` (каналы, высота, ширина). После `permute(1, 2, 0)` форма становится `[H, W, C]`. Например, если мы имеем тензор с формой `(3, 600, 600)`, то на выходе получаем тензор с формой `(600, 600, 3)`, который удобен для отображения с помощью matplotlib.
    
7. `plt.imshow(img)`: отображаем изображение на основе данных
    
    Если изображение имеет лишние размерности равные 1, то для их удаления можно использовать функицию `squeeze()`:

```python
plt.imshow(img.squeeze())
```

### Загрузка и итерация данных с помощью загрузчика

Аналогично мы можем использовать загрузчики данных (dataloader) для итерации по загруженному набору данных. Например:

```python

import torch
from torch.utils.data import Dataset, DataLoader
from torchvision.io import decode_image
import matplotlib.pyplot as plt
import pandas as pd
 
class MyImageDataset(Dataset):
    def __init__(self, labels_file, img_dir, transform=None, target_transform=None):
        self.img_labels = pd.read_csv(labels_file)
        self.img_dir = img_dir
        self.transform = transform
        self.target_transform = target_transform
 
    def __len__(self):
        return len(self.img_labels)
 
    def __getitem__(self, idx):
        img_path = os.path.join(self.img_dir, self.img_labels.iloc[idx, 0])
        image = decode_image(img_path)
        label = self.img_labels.iloc[idx, 1]
        if self.transform:
            image = self.transform(image)
        if self.target_transform:
            label = self.target_transform(label)
        return image, label
 
 
# набор данных
data = MyImageDataset("labels.csv", "images")
  
labels_map = {
    0: "Пальто",
    1: "Рубашка",
    2: "Свитер",
    3: "Штаны",
    4: "Футболка",
}
 
# загружаем по 3 изображения
dataloader = DataLoader(data, batch_size=3, shuffle=True)
# отображаем картинку и метку
images, labels = next(iter(dataloader))
print(f"Форма изображений батча: {images.size()}")
print(f"Количество меток: {labels.size()}")
img = images[0].squeeze()   # берем первое изображение и убираем лишнюю размерность
label = labels[0]   # берем первую метку
img = img.permute(1, 2, 0)  # переупорядочиваем размерности изображения
plt.title(labels_map[label.item()])     # устанавливаем заголовок графика
plt.imshow(img)     # устанавливаем изображение для отображения
plt.show()  # отображаем график
```

В данном случае загружаем с помощью загрузчика одномоментно 3 изображения, выводим на консоль информацию об этих изоражениях и их метках. А затем берем первое изображение и его метку и отображаем его.

### Преобразования тензоров

При работе с тензорами могут потребоваться различные преобразования. Например, в примере выше применялось следующее преобразование:

```python
img = img.permute(1, 2, 0)
```

которое переупорядочивает размерности. То есть, например, из тензора с формой `torch.Size([3, 600, 600])` позволяет получить тензор с формой `torch.Size([600, 600, 3])`, то есть переместить значение цветового канала в конец.

Другое использованное преобразование

```python
img = img.squeeze()
```

Это преобразование убирает единичные размерности. То есть, например, из тензора с формой `torch.Size([1, 600, 600])` позволяет получить тензор с формой `torch.Size([600, 600])`.

Но рассмотрим еще несколько приемов, которые могут потребоваться при работе с тензорами изображений. Так, метод `squeeze()` убирает единичные размерности, но если размерность равна, например, 3 или 4, то метод не окажет никакого влияния. Но, например, нам надо из тензора с формой `torch.Size([3, 600, 600])` получить тензор с формой `torch.Size([600, 600])`, то есть убрать цветовой канал и оставить только ширину и высоту. Для этого можно использовать

```python
tensor_600_600 = img[0] 
```

В данном случае мы делаем срез по первой размерности, сохраняя все остальные размерности (получаем `torch.Size([600, 600])`).

Либо, например, мы можем взять максимальные значения:

```python
tensor_600_600 = img.max(dim=0)[0] # если нужен максимум
```

`dim=0` означает, что мы ищем максимум по первому измерению. Для каждой позиции (i, j) в размерах `[600, 600]` функция сравнивает 3 значения из разных измерений и возвращает максимальное значение для каждой позиции. Затем извлекает значения из кортежа результата - `torch.max()` возвращает кортеж из двух тензоров: `[0]` - тензор с максимальными значениями и `[1]` - тензор с индексами, где были найдены максимумы

Другое распространенное преобразование - это преобразование между типами данных. Так, для обучения и распознавания может потребоваться тензор с данными типа `float`, но функция `decode_image` возвращает тензор с целочисленными значениями `uint8`. Для преобразования из целых чисел в значения с плавающей точкой можно использовать функцию torchvision.transforms.v2.functional.to_dtype(), в которую передается преобразуемый тензор:

```python
from torchvision.transforms import v2
......................................
image = v2.functional.to_dtype(image)  # int -> float
```

Приведем ориентировочный пример с переносом преобразований непосредственно в код датасета:

```python
import os
import torch
from torch.utils.data import Dataset, DataLoader
from torchvision.io import decode_image
from torchvision.transforms import v2
import matplotlib.pyplot as plt
import pandas as pd
 
class MyImageDataset(Dataset):
    def __init__(self, labels_file, img_dir, transform=None, target_transform=None):
        self.img_labels = pd.read_csv(labels_file)
        self.img_dir = img_dir
        self.transform = transform
        self.target_transform = target_transform
 
    def __len__(self):
        return len(self.img_labels)
 
    def __getitem__(self, idx):
        img_path = os.path.join(self.img_dir, self.img_labels.iloc[idx, 0])
        image = decode_image(img_path)
 
        image = image[0]  # torch.Size([3, 600, 600]) -> torch.Size([600, 600])
        image = v2.functional.to_dtype(image)  # int -> float
 
        label = self.img_labels.iloc[idx, 1]
        if self.transform:
            image = self.transform(image)
        if self.target_transform:
            label = self.target_transform(label)
        return image, label
 
 
# набор данных
data = MyImageDataset("labels.csv", "images")
  
dataloader = DataLoader(data, batch_size=2, shuffle=True)
  
# отображаем картинку
features, labels = next(iter(dataloader))
img = features[0]  # никаких преобразований не нужно, тензор представляет двухмерное изображение
plt.imshow(img)
plt.show()
```
