В предыдущих статьях мы рассмотрели, как обучить модель (нейронную сеть) на основе встроенного набора FashionMNIST для распознавания изображений с одеждой. Сейчас же рассмотрим, как мы можем распознавать произвольные изображения. Поскольку в нашем случае сеть обучена картинках с одеждой и предназначена для распознавания элементов одежды, то соответственно мы будем распознавать также изображение с некоторым элементом одежды.

Будем предполагать, что модель уже обучена, и что ее веса сохранены в файл model_weights.pth, который находится в той же папке, что и текущий скрипт. Для распознавания изображений определим следующий код:

```python
import torch
from torch import nn
from torchvision import transforms
from PIL import Image # Импортируем Pillow
 
# Определяем класс
class NeuralNetwork(nn.Module):
    def __init__(self):
        super().__init__()
        self.flatten = nn.Flatten()
        self.linear_relu_stack = nn.Sequential(
            nn.Linear(28*28, 512),
            nn.ReLU(),
            nn.Linear(512, 512),
            nn.ReLU(),
            nn.Linear(512, 10),
        )
 
    def forward(self, x):
        x = self.flatten(x)
        logits = self.linear_relu_stack(x)
        return logits
 
# Создаем "пустую" модель
model = NeuralNetwork()
 
# Загружаем веса из файла
# Убедитесь, что файл "model_weights.pth" находится в той же папке
model.load_state_dict(torch.load("model_weights.pth"))
 
# Переключаем в режим оценки
model.eval()
 
print("Модель загружена!")
 
# Список классов FashionMNIST для расшифровки ответа: локализация следующих классов
#classes = [
#    "T-shirt/top", "Trouser", "Pullover", "Dress", "Coat",
#    "Sandal", "Shirt", "Sneaker", "Bag", "Ankle boot",
#]
classes = [
    "Футболка/топ", "Брюки", "Свитер", "Платье", "Пальто",
    "Сандалии", "Рубашка", "Кроссовки", "Сумка", "Ботинки",
]
 
# Путь к изображению
IMAGE_PATH = "tshirt.jpeg" # ЗАМЕНИТЕ НА ВАШ ФАЙЛ
 
# Определяем преобразования
# Модель обучена на FashionMNIST, который имеет следующие характеристики:
# - Размер: 28x28
# - Цветовой канал: 1 (черно-белое / оттенки серого)
# - Тип данных: float от 0.0 до 1.0 (стандартизированный)
# 
# Шаги: Загрузка изображения -> Масштабирование до размера 28x28 -> Преобразование в серый -> Преобразуем в тензор (0-1 float)
 
image_transform = transforms.Compose([
    transforms.Resize((28, 28)), # Изменяем размер до 28x28
    transforms.Grayscale(num_output_channels=1), # Преобразуем в оттенки серого (1 канал)
    transforms.ToTensor(), # Преобразует в Tensor и нормализует в диапазон [0, 1]
])
 
# Загружаем изображение с помощью Pillow
image = Image.open(IMAGE_PATH)
 
# Применяем преобразования
x = image_transform(image)
# Теперь x имеет размерность [1, 28, 28]
 
# --- ПРЕДСКАЗАНИЕ ---
# Добавляем к тензору изображения размерность батча (было [1, 28, 28] -> стало [1, 1, 28, 28])
with torch.no_grad():
    pred = model(x.unsqueeze(0)) # Здесь x - наше преобразованное изображение
    predicted_class_index = pred[0].argmax(0)
    predicted_class_name = classes[predicted_class_index]
 
print(f"\nИзображение: {IMAGE_PATH}")
print(f"Предсказанный класс: {predicted_class_name}")


Модель загружена!
Изображение: tshirt.jpeg
Предсказанный класс: Рубашка
```

Рассмотрим основные моменты. Итак, чтобы использовать реальное изображение (например, файл png или jpg), нам нужно выполнить несколько шагов:

1. Загрузить изображение с диска
    
2. Преобразовать изображение в формат, который ожидает наша модель PyTorch
    

Для загрузки изображения нам понадобится библиотека Pillow (PIL). Если эта библиотека вдруг не установлена, то ее можно установить командой `pip install Pillow`. Соответственно вначале скрипта подключаем библиотеку Pillow:

```python
from PIL import Image
```

Следующий важный момент - определение преобразований в виде переменной `image_transform`

```python
image_transform = transforms.Compose([
    transforms.Resize((28, 28)), # Изменяем размер до 28x28
    transforms.Grayscale(num_output_channels=1), # Преобразуем в оттенки серого (1 канал)
    transforms.ToTensor(), # Преобразует в Tensor и нормализует в диапазон [0, 1]
])
```

При работе с моделью надо учитывать, что она обучена на FashionMNIST, который имеет следующие характеристики: размер 28x28, 1 цветовой канал (черно-белое / оттенки серого) и тип данных - `float` с диапазоном значений от 0.0 до 1.0. Поэтому нам надо определить соответствующие действия для преобразования изображения в нужный формат. Поэтому здесь мы создаем объект `transforms.Compose` для последовательного выполнения шагов, которые сделают наше изображение совместимым с моделью:

1. `transforms.Resize((28, 28))`: уменьшает или увеличивает изображение до нужного размера 28x28
    
2. `transforms.Grayscale(num_output_channels=1)`: преобразует цветное изображение в черно-белое (1 канал), так как FashionMNIST состоит из одноканальных изображений
    
3. `transforms.ToTensor()`: конвертирует изображение PIL в тензор PyTorch и автоматически переводит значения пикселей в диапазон `[0.0, 1.0]`
    

Далее загружаем изображение и применяем к нему преобразования, чтобы на выходе получить тензор, совместимый с нашей моделью:

```python
image = Image.open(IMAGE_PATH)
x = image_transform(image)
```

В конце, как уже было рассмотрено в прошлых темах, передаем тензор в модель и получаем предсказание модели и связанную с ним текстовую метку:

```python
with torch.no_grad():
    pred = model(x.unsqueeze(0)) # Здесь x - наше преобразованное изображение
    predicted_class_index = pred[0].argmax(0)
    predicted_class_name = classes[predicted_class_index]
```
